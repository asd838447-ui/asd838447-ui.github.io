Билет 1. Принцип математической индукции. Бином Ньютона. Неравенство Бернулли.

Для ответа на этот экзаменационный вопрос мы последовательно разберем принцип математической индукции как метод доказательства, а затем с его помощью выведем формулу бинома Ньютона и докажем неравенство Бернулли.

1. Принцип математической индукции

Принцип математической индукции — это фундаментальный метод доказательства утверждений, зависящих от натурального числа \( n \). Суть метода заключается в том, что утверждение считается истинным для всех натуральных чисел, если выполняются два строгих условия:

  • База индукции: Утверждение проверяется и признается верным для начального значения (обычно для \( n = 1 \)).
  • Индукционный переход: Из предположения о том, что утверждение верно для некоторого произвольного числа \( k \) (предположение индукции), строго математически следует его истинность для следующего за ним числа \( k+1 \).

Если оба этих шага выполнены, то, согласно аксиоме индукции, формула или свойство доказано для любого натурального \( n \).

2. Бином Ньютона

Формула бинома Ньютона описывает разложение произвольной натуральной степени двучлена \( (a+b)^n \) в виде многочлена.

Основные понятия: факториал и биномиальные коэффициенты

Для правильной записи формулы нам необходимо определить два важных понятия комбинаторики:

  • Факториал числа \( n \) (обозначается \( n! \)) — это произведение всех натуральных чисел от 1 до \( n \): \[ n! = 1 \cdot 2 \cdot 3 \cdot \dots \cdot n \] По определению полагают, что \( 0! = 1 \).
  • Биномиальный коэффициент (\( C_n^k \) или \( \binom{n}{k} \)) определяет количество конфигураций — число способов выбрать \( k \) элементов из \( n \). Вычисляется по строгой формуле: \[ C_n^k = \frac{n!}{k!(n-k)!} \]

Формула разложения (Теорема о Биноме)

Общий вид формулы бинома Ньютона выглядит следующим образом:

\[ (a+b)^n = \sum_{k=0}^{n} C_n^k a^{n-k} b^k = C_n^0 a^n + C_n^1 a^{n-1} b + C_n^2 a^{n-2} b^2 + \dots + C_n^n b^n \]

Важной логической особенностью является то, что в каждом слагаемом многочлена сумма показателей степеней множителей \( a \) и \( b \) всегда равна \( n \). Коэффициенты разложения можно также находить графически с помощью треугольника Паскаля, где каждое число равно сумме двух чисел, стоящих строго над ним в предыдущей строке.

Доказательство методом математической индукции

Докажем формулу: \( (a+b)^n = \sum_{k=0}^{n} C_n^k a^{n-k} b^k \).

  • База индукции (\( n = 1 \)): Для левой части \( (a+b)^1 = a+b \). Развернём правую часть: \( C_1^0 a^1 b^0 + C_1^1 a^0 b^1 = 1 \cdot a \cdot 1 + 1 \cdot 1 \cdot b = a+b \). База доказана — утверждение верно.
  • Предположение индукции: Пусть формула верна для \( n = m \), то есть: \[ (a+b)^m = C_m^0 a^m + C_m^1 a^{m-1} b + \dots + C_m^m b^m \]
  • Индукционный переход (\( n = m+1 \)): Рассмотрим выражение для следующей степени: \[ (a+b)^{m+1} = (a+b)^m (a+b) \] Подставим наше предположение для \( (a+b)^m \) и раскроем скобки, умножая выражение сначала на \( a \), а затем на \( b \): \[ = \Big(C_m^0 a^{m+1} + C_m^1 a^m b + \dots + C_m^m a b^m\Big) + \Big(C_m^0 a^m b + C_m^1 a^{m-1} b^2 + \dots + C_m^m b^{m+1}\Big) \] Приведем подобные слагаемые при одинаковых степенях \( a \) и \( b \): \[ = C_m^0 a^{m+1} + (C_m^0 + C_m^1) a^m b + (C_m^1 + C_m^2) a^{m-1} b^2 + \dots + C_m^m b^{m+1} \] Используя ключевую лемму о свойстве биномиальных коэффициентов \( C_m^{k-1} + C_m^k = C_{m+1}^k \), а также граничные факты, что \( C_m^0 = C_{m+1}^0 = 1 \) и \( C_m^m = C_{m+1}^{m+1} = 1 \), мы приходим к финальному виду: \[ (a+b)^{m+1} = C_{m+1}^0 a^{m+1} + C_{m+1}^1 a^m b + \dots + C_{m+1}^{m+1} b^{m+1} \] Формула доказана для \( n = m+1 \), а значит, по фундаментальному принципу индукции, она верна для любого натурального значения \( n \).

3. Неравенство Бернулли

Неравенство Бернулли утверждает, что для любого действительного числа \( a \geq -1 \) и любого натурального \( n \in \mathbb{N} \) всегда справедливо следующее соотношение:

\[ (1 + a)^n \geq 1 + na \]

Доказательство методом математической индукции

  • База индукции (\( n = 1 \)): Подставим \( n = 1 \) в уравнение: \( (1+a)^1 \geq 1 + 1 \cdot a \), что математически сводится к \( 1+a \geq 1+a \). Это истинное нестрогое равенство, база доказана.
  • Предположение индукции: Пусть неравенство абсолютно верно для некоторого \( n = k \), то есть: \( (1+a)^k \geq 1 + ka \).
  • Индукционный переход (\( n = k+1 \)): Рассмотрим левое выражение для следующей степени: \[ (1+a)^{k+1} = (1+a)^k (1+a) \] Так как по исходному условию теоремы \( a \geq -1 \), то множитель \( (1+a) \geq 0 \). Это важное свойство позволяет нам умножить обе части неравенства из предположения индукции на \( (1+a) \), не меняя при этом знак самого неравенства: \[ (1+a)^k (1+a) \geq (1 + ka)(1 + a) \] Теперь раскроем скобки в получившейся правой части: \[ (1 + ka)(1 + a) = 1 + a + ka + ka^2 = 1 + (k+1)a + ka^2 \] Поскольку величина \( ka^2 \) гарантированно неотрицательна (так как \( k \) — натуральное число, \( a^2 \geq 0 \)), мы имеем право просто отбросить это слагаемое, тем самым лишь уменьшив правую часть выражения: \[ 1 + (k+1)a + ka^2 \geq 1 + (k+1)a \] Объединяя шаги в общую логическую цепочку, мы получаем искомый результат: \[ (1+a)^{k+1} \geq 1 + (k+1)a \] Индукционный переход строго доказан, а следовательно, Неравенство Бернулли абсолютно верно для всех \( n \in \mathbb{N} \).
