Билет 3. Модуль числа, два простых неравенства с модулем. Системы вложенных отрезков, стягивающиеся системы вложенных отрезков, две теоремы об этих системах.

1. Модуль числа и его свойства

Модуль (абсолютная величина) действительного числа a (обозначается |a|) — это само число a, если оно неотрицательно, и число −a (противоположное), если a отрицательно. Формально это записывается так:

|a| = { a, если a≥0; −a, если a<0 }

Геометрически модуль отражает расстояние на числовой прямой от начала координат до точки, соответствующей числу a. Из определения следует, что модуль всегда неотрицателен (|a|≥0), а также справедливы соотношения: |a| = max(−a, a), |−a| = |a|, −a ≤ |a| и a ≤ |a|.

Два основных неравенства с модулем (Лемма о свойствах модуля)

В анализе наиболее часто используются следующие два неравенства для любых чисел a и b:

Неравенство 1 (Неравенство треугольника):

|a+b| ≤ |a| + |b|

Доказательство: На основании свойств действительных чисел можно записать: a ≤ |a| и b ≤ |b|. Складывая их, получаем a+b ≤ |a|+|b|. Аналогично, из −a ≤ |a| и −b ≤ |b| следует −(a+b) ≤ |a|+|b|. Так как обе величины (a+b) и −(a+b) не превосходят суммы модулей, то и их максимум (которым является |a+b|) также не превосходит этой суммы: max(−(a+b), a+b) ≤ |a|+|b|.

Неравенство 2 (Неравенство для разности модулей):

||a| − |b|| ≤ |a−b|

Доказательство: Представим число a в виде a = (a−b)+b. Применяя первое неравенство (неравенство треугольника), получим: |a| ≤ |a−b| + |b|. Перенося |b| в левую часть, имеем: |a| − |b| ≤ |a−b|. Теперь представим b как b = (b−a)+a. Аналогично: |b| ≤ |b−a| + |a| ⇒ |b| − |a| ≤ |b−a|. Учитывая, что |a−b| = |b−a|, второе неравенство можно переписать как −(|a|−|b|) ≤ |a−b|. Поскольку и (|a|−|b|), и −(|a|−|b|) не превышают |a−b|, то их максимум (величина ||a|−|b||) также удовлетворяет этому условию: max(|a|−|b|, −(|a|−|b|)) ≤ |a−b|.

2. Системы вложенных отрезков

Определение: Последовательность отрезков {[aₙ, bₙ]} называется системой вложенных отрезков (СВО), если каждый последующий отрезок полностью содержится в предыдущем. Математически это записывается как:

[a₁, b₁] ⊃ [a₂, b₂] ⊃ [a₃, b₃] ⊃ ... ⊃ [aₙ, bₙ] ⊃ ...

Это условие эквивалентно выполнению неравенств aₙ ≤ aₙ₊₁ и bₙ₊₁ ≤ bₙ для всех натуральных n. Из этого следует, что последовательность левых концов {aₙ} является неубывающей, а правых концов {bₙ} — невозрастающей, причем любой левый конец никогда не превосходит любого правого конца (aₖ ≤ bₙ для любых индексов k, n).

3. Стягивающиеся системы вложенных отрезков

Определение: Система вложенных отрезков {[aₙ, bₙ]} называется стягивающейся, если длины этих отрезков стремятся к нулю при неограниченном росте n. Это записывается как:

lim (bₙ − aₙ) = 0 при n→∞

Иными словами, для любого сколь угодно малого числа ε>0 найдется такой номер nₑ, что длина отрезка с этим номером (и всех последующих) будет меньше ε.

4. Две фундаментальные теоремы о вложенных отрезках

Существование действительных чисел и многие свойства непрерывности базируются на двух следующих теоремах.

Теорема 1 (Принцип вложенных отрезков Кантора)

Формулировка: Всякая система вложенных отрезков имеет по крайней мере одну общую точку (непустое пересечение).

Доказательство:

  1. Рассмотрим множество всех левых концов A = {aₙ} и множество всех правых концов B = {bₙ}.
  2. Как было отмечено ранее, любой элемент из A не превосходит любого элемента из B (aₖ ≤ bₙ).
  3. Согласно аксиоме (свойству) непрерывности действительных чисел, если для двух непустых множеств X и Y выполняется условие x ≤ y для всех их элементов, то существует такое число c, что x ≤ c ≤ y.
  4. В нашем случае это означает существование числа c, такого что aₙ ≤ c ≤ bₙ для всех натуральных n.
  5. Неравенство aₙ ≤ c ≤ bₙ по определению означает, что точка c принадлежит каждому отрезку [aₙ, bₙ] данной системы. Таким образом, общая точка существует.

Теорема 2 (Единственность точки для стягивающейся системы)

Формулировка: Если система вложенных отрезков является стягивающейся, то существует только одна точка, принадлежащая всем отрезкам системы одновременно.

Доказательство:

  1. Предположим противное: пусть существуют две различные общие точки c и d. Для определенности пусть c < d.
  2. Поскольку обе точки принадлежат всем отрезкам, то для каждого n выполняются условия aₙ ≤ c и d ≤ bₙ. Отсюда следует, что расстояние между точками |c−d| не может превышать длины любого из отрезков: 0 < d−c ≤ bₙ − aₙ.
  3. По условию стягиваемости системы, длина отрезков bₙ − aₙ стремится к нулю при n→∞.
  4. Применяя теорему о «зажатой» последовательности (теорему о двух милиционерах) к неравенству 0 ≤ |d−c| ≤ bₙ − aₙ, получаем, что предел |d−c| при n→∞ должен быть равен нулю.
  5. Но так как c и d — фиксированные числа, их разность может быть равна нулю только если c = d. Это противоречит предположению о наличии двух разных точек. Следовательно, общая точка единственна.

Важное замечание: Эти теоремы верны только для замкнутых промежутков (отрезков). Для систем вложенных интервалов (открытых промежутков) общая точка может отсутствовать, что подчеркивает важность включения концов промежутков в рассмотрение.
