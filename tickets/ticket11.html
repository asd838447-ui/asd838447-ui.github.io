Билет 11. Тригонометрическая форма комплексного числа. Корень натуральной степени комплексного числа. Решение квадратных уравнений над полем комплексных чисел.

1. Тригонометрическая форма комплексного числа

Комплексные числа являются расширением множества действительных чисел R, позволяющим решать уравнения, не имеющие действительных корней. Каждое комплексное число z в базовой алгебраической форме записывается как z = a + bi, где a — действительная часть (Re z), а b — мнимая часть (Im z).

Для перехода к тригонометрической форме необходимо рассмотреть геометрическое представление числа на комплексной плоскости (плоскости Аргана), где ось Ox является действительной, а ось Oy — мнимой. В этой системе число z = a + bi представляется как точка с координатами (a, b) или как радиус-вектор, проведенный из начала координат в эту точку.

Основные характеристики:

  • Модуль комплексного числа |z| (обозначается также латинской буквой r) — это длина радиус-вектора, которая по теореме Пифагора вычисляется как r = |z| = √(a² + b²). Модуль всегда является неотрицательным действительным числом и отражает расстояние от начала координат до точки.
  • Аргумент комплексного числа ϕ (обозначается Arg z) — это совокупность всех углов, которые образует вектор z с положительным направлением действительной оси. Поскольку тригонометрические функции периодичны, аргумент определен с точностью до 2πk, где k ∈ Z. Значение аргумента, лежащее в полуинтервале [0, 2π) или (−π, π], называется главным значением аргумента и обозначается arg z.

Связь координат: Из определений синуса и косинуса в прямоугольном треугольнике на комплексной плоскости следуют соотношения:

  • a = r cos ϕ
  • b = r sin ϕ

Подставив эти выражения в алгебраическую форму z = a + bi, мы получаем тригонометрическую форму записи комплексного числа:

z = r (cos ϕ + i sin ϕ)

Эта форма особенно удобна для операций умножения и деления:

  • При умножении двух комплексных чисел их модули перемножаются, а аргументы складываются: z₁·z₂ = r₁ r₂ (cos(ϕ + ψ) + i sin(ϕ + ψ)).
  • При делении модули делятся, а аргументы вычитаются: z₁ / z₂ = (r₁ / r₂) (cos(ϕ − ψ) + i sin(ϕ − ψ)).

2. Корень натуральной степени из комплексного числа

Пусть дано комплексное число z и натуральное число n ∈ N, n > 1. Определение: Число w называется корнем n-й степени из числа z, если выполняется равенство wⁿ = z.

Для нахождения всех значений корня удобнее всего использовать тригонометрическую форму записи. Пусть z = r (cos ϕ + i sin ϕ), а искомый корень w = ρ (cos θ + i sin θ). Согласно формуле Муавра (возведение в степень): wⁿ = ρⁿ (cos(nθ) + i sin(nθ)). Приравнивая wⁿ к z, получаем систему:

  • ρⁿ = r ⇒ ρ = ⁿ√r (арифметический корень из положительного действительного числа).
  • nθ = ϕ + 2πk ⇒ θₖ = (ϕ + 2πk)/n, где k ∈ Z.

Чтобы получить все различные значения корня, достаточно перебрать n последовательных значений индекса k, обычно k = 0, 1, 2, …, n−1. При других значениях k аргументы будут отличаться на числа, кратные 2π, и значения корней начнут повторяться из-за периодичности синуса и косинуса.

Общая формула корней n-й степени:

wₖ = ⁿ√r (cos((ϕ + 2πk)/n) + i sin((ϕ + 2πk)/n)), k = 0, 1, …, n−1

Геометрический смысл: Все n корней степени n из комплексного числа z расположены на окружности радиуса R = ⁿ√r с центром в начале координат. Они являются вершинами правильного n-угольника, вписанного в эту окружность. Первый корень соответствует углу ϕ/n, а каждый последующий повернут относительно предыдущего на угол 2π/n.

3. Решение квадратных уравнений над полем комплексных чисел

Квадратное уравнение имеет вид ax² + bx + c = 0, где a, b, c — коэффициенты (в общем случае комплексные, но часто рассматриваются действительные), при этом a ≠ 0. Согласно основной теореме алгебры, любой многочлен степени n имеет ровно n комплексных корней с учетом их кратности. Таким образом, квадратное уравнение всегда имеет ровно два корня в поле C.

Для решения используется дискриминант D = b² − 4ac. Рассмотрим случаи для уравнения с действительными коэффициентами:

  • D > 0: Уравнение имеет два различных действительных корня, вычисляемых по стандартной формуле x₁,₂ = (−b ± √D) / (2a).
  • D = 0: Уравнение имеет один действительный корень кратности 2 (два совпадающих корня): x₁ = x₂ = −b/(2a).
  • D < 0: В поле действительных чисел корней нет, но в поле комплексных чисел они существуют.

Механизм решения при D < 0:

Поскольку D отрицательно, мы можем представить его как D = |D|·(−1). Используя фундаментальное свойство мнимой единицы i² = −1, записываем D = |D|·i². Тогда извлечение корня из отрицательного дискриминанта дает:

√D = √(|D|·i²) = ± i √|D|

Формула комплексных корней:

x₁,₂ = (−b ± i √|D|) / (2a)

Эти корни являются комплексно-сопряженными числами вида x₁ = α + βi и x₂ = α − βi. Важно отметить, что в поле комплексных чисел операция извлечения квадратного корня всегда дает два значения, поэтому даже если D — это отрицательное число, мы получаем два полноценных решения.

В случае, если сами коэффициенты a, b, c являются комплексными числами, для извлечения √D используется общая формула корня n-й степени из комплексного числа (описанная выше для n = 2).
