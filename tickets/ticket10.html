Билет 10. Комплексные числа. Алгебраическая и тригонометрическая формы записи, арифметические операции на поле комплексных чисел. Формула Муавра.

В данном ответе подробно рассмотрено понятие комплексных чисел, их формы записи, правила выполнения арифметических операций и доказательство формулы Муавра на основе предоставленных источников.

1. Определение комплексных чисел и мнимая единица

Комплексные числа представляют собой расширение множества действительных чисел (R), которое позволяет решать уравнения, не имеющие корней в R (например, x² + 1 = 0).

Мнимая единица — это число, обозначаемое символом i, для которого справедливо фундаментальное соотношение:

i² = −1

Комплексные числа образуют множество C. Каждое комплексное число z можно представить как упорядоченную пару действительных чисел (a, b).

2. Алгебраическая форма записи

Алгебраическая форма является базовой и наиболее интуитивно понятной формой записи комплексного числа:

z = a + bi

Где:

  • a = Re z — действительная часть (от лат. res — вещь, реальность).
  • b = Im z — мнимая часть (от лат. imago — изображение, мнимость).

Если a = 0, число называется чисто мнимым (например, 2i).

Если b = 0, число совпадает с обычным действительным числом. Таким образом, R ⊂ C.

Комплексно сопряженное число к числу z = a + bi обозначается \bar{z} и определяется как:

\bar{z} = a - bi

Важным свойством является то, что произведение числа на его сопряженное дает сумму квадратов его частей:

z · \bar{z} = a² + b²

Модуль комплексного числа |z| — это расстояние от начала координат до точки (a, b) на комплексной плоскости:

|z| = √(a² + b²)

3. Арифметические операции в алгебраической форме

Пусть даны два комплексных числа: z₁ = a₁ + b₁i и z₂ = a₂ + b₂i.

Сложение:

z₁ + z₂ = (a₁ + a₂) + (b₁ + b₂)i

Выполняется поэлементно (отдельно складываются действительные и мнимые части).

Вычитание:

z₁ − z₂ = (a₁ − a₂) + (b₁ − b₂)i

Выполняется аналогично сложению.

Умножение:

z₁ · z₂ = (a₁ + b₁i)(a₂ + b₂i) = a₁a₂ + a₁b₂i + b₁a₂i + b₁b₂i² = (a₁a₂ − b₁b₂) + (a₁b₂ + b₁a₂)i

Выполняется по правилу раскрытия скобок с учетом того, что i² = −1.

Деление:

Чтобы разделить z₁ на z₂ (z₂ ≠ 0), необходимо умножить числитель и знаменатель на сопряженное к знаменателю число \bar{z₂} = a₂ − b₂i. Это позволяет избавиться от мнимости в знаменателе:

z₁ / z₂ = (a₁ + b₁i)(a₂ − b₂i) / ((a₂ + b₂i)(a₂ − b₂i)) = ((a₁a₂ + b₁b₂) + (b₁a₂ − a₁b₂)i) / (a₂² + b₂²)

4. Тригонометрическая форма записи

Любое комплексное число z ≠ 0 можно представить через его модуль r = |z| и аргумент ϕ. Аргумент ϕ (обозначается Arg z) — это угол между положительным направлением действительной оси и вектором, проведенным из начала координат к точке z.

Связь между координатами (a, b) и (r, ϕ):

  • a = r cos ϕ
  • b = r sin ϕ

Тригонометрическая форма:

z = r (cos ϕ + i sin ϕ)

Эта форма особенно удобна для операций умножения и деления:

Умножение в тригонометрической форме:

z₁ · z₂ = r₁ r₂ (cos(ϕ + ψ) + i sin(ϕ + ψ))

Модули перемножаются, а аргументы складываются.

Деление в тригонометрической форме:

z₁ / z₂ = (r₁ / r₂) (cos(ϕ − ψ) + i sin(ϕ − ψ))

Модули делятся, а аргументы вычитаются.

5. Формула Муавра

Формула Муавра используется для возведения комплексного числа, записанного в тригонометрической форме, в натуральную степень n ∈ N.

Формулировка:

(cos ϕ + i sin ϕ)ⁿ = cos(nϕ) + i sin(nϕ)

Доказательство (через формулу Эйлера):

  1. Согласно формуле Эйлера, cos ϕ + i sin ϕ = e^{iϕ}.
  2. Возведем обе части этого равенства в степень n: (cos ϕ + i sin ϕ)ⁿ = (e^{iϕ})ⁿ.
  3. По свойству степеней, при возведении степени в степень показатели перемножаются: (e^{iϕ})ⁿ = e^{i(nϕ)}.
  4. Применяя формулу Эйлера обратно к выражению e^{i(nϕ)}, где аргументом теперь является nϕ, получаем: e^{i(nϕ)} = cos(nϕ) + i sin(nϕ).

Таким образом, доказано: (cos ϕ + i sin ϕ)ⁿ = cos(nϕ) + i sin(nϕ).

Для общего случая zⁿ, где z = r (cos ϕ + i sin ϕ), формула принимает вид:

zⁿ = rⁿ (cos(nϕ) + i sin(nϕ))
