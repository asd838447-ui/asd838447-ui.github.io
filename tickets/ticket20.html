🔹 Эквивалентности (при x → 0)

sin x ∼ x
arcsin x ∼ x
tan x ∼ x
arctan x ∼ x
eˣ − 1 ∼ x
ln(1 + x) ∼ x
aˣ − 1 ∼ x·ln a
(1 + x)ᵏ − 1 ∼ kx
1 − cos x ∼ x²/2
sh x ∼ x
th x ∼ x
ch x − 1 ∼ x²/2

🔹 Обобщённые эквивалентности (при u(x) → 0)

sinᵇ u ∼ uᵇ
arcsinᵇ u ∼ uᵇ
tanᵇ u ∼ uᵇ
arctanᵇ u ∼ uᵇ
(eᵘ − 1)ᵇ ∼ uᵇ
lnᵇ(1 + u) ∼ uᵇ
(1 + u)ᵏ − 1 ∼ k·u
1 − cosᵇ u ∼ (b/2)·u²
shᵇ u ∼ uᵇ
thᵇ u ∼ uᵇ

🔹 Ряды Тейлора (при x → 0)

eˣ = 1 + x + x²/2! + x³/3! + o(x³)
sin x = x − x³/3! + x⁵/5! − x⁷/7! + o(x⁷)
cos x = 1 − x²/2! + x⁴/4! − x⁶/6! + o(x⁶)
tan x = x + x³/3 + 2x⁵/15 + o(x⁵)
arcsin x = x + x³/6 + 3x⁵/40 + o(x⁵)
arctan x = x − x³/3 + x⁵/5 − x⁷/7 + o(x⁷)
ln(1 + x) = x − x²/2 + x³/3 − x⁴/4 + o(x⁴)
(1 + x)ᵅ = 1 + αx + α(α−1)x²/2! + o(x²)
sh x = x + x³/3! + x⁵/5! + x⁷/7! + o(x⁷)
ch x = 1 + x²/2! + x⁴/4! + x⁶/6! + o(x⁶)

🔹 Производные элементарных функций

(xⁿ)' = n·xⁿ⁻¹
(sin x)' = cos x
(cos x)' = −sin x
(tan x)' = 1/cos²x = 1 + tan²x
(cot x)' = −1/sin²x = −(1 + cot²x)
(eˣ)' = eˣ
(aˣ)' = aˣ·ln a
(ln x)' = 1/x
(log_a x)' = 1/(x·ln a)
(arcsin x)' = 1/√(1−x²)
(arccos x)' = −1/√(1−x²)
(arctan x)' = 1/(1+x²)
(sh x)' = ch x
(ch x)' = sh x
(th x)' = 1/ch²x

🔹 Первый и второй замечательные пределы

lim_{x→0} sin x / x = 1
lim_{x→0} (1 + x)^{1/x} = e
lim_{x→∞} (1 + 1/x)ˣ = e

🔹 Тригонометрические тождества

sin²x + cos²x = 1
1 + tan²x = 1/cos²x
1 + cot²x = 1/sin²x
sin(α ± β) = sinα·cosβ ± cosα·sinβ
cos(α ± β) = cosα·cosβ ∓ sinα·sinβ
sin2x = 2 sinx·cosx
cos2x = cos²x − sin²x = 2cos²x − 1 = 1 − 2sin²x
sin²x = (1 − cos2x)/2
cos²x = (1 + cos2x)/2
tan2x = 2tanx / (1 − tan²x)
