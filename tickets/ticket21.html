📚 Теория в контексте

Изучайте свойства функций не по учебнику, а на живом примере. Измените формулу выше — и теория обновится автоматически!

🔹 Область определения

Множество значений x, при которых функция имеет смысл.

D(f) = ℝ
Для 1/x: x ≠ 0
Для √x: x ≥ 0
Для log(x): x > 0

🔹 Область значений

Множество всех возможных результатов y = f(x).

E(f) = ℝ
sin(x): [-1; 1]
x²: [0; +∞)
e^x: (0; +∞)

🔹 Асимптоты

Прямые, к которым график неограниченно приближается.

Вертикальная: x = a, если f(x)→±∞ при x→a
Горизонтальная: y = b, если f(x)→b при x→±∞
Наклонная: y = kx + m, где k = lim f(x)/x

🔹 Чётность и периодичность

Чётная: f(-x) = f(x) → симметрия относительно Oy
Нечётная: f(-x) = -f(x) → симметрия относительно начала
Периодическая: f(x+T) = f(x), T — период
sin(x), cos(x): T = 2π  |  tan(x): T = π
