🎓 Билет 21: График функций

Введите формулу → получите график + теорию. Работает без интернета. Понятно даже новичку.

Поддерживается: + - * / ^ sin cos tan log ln exp sqrt abs | Переменная: x
x² Парабола
sin(x) Синус
cos(x) Косинус
e⁻ˣ² Колокол
lg(x) Логарифм
1/x Гипербола
|x| Модуль
x³-3x Кубическая

🔍 Масштаб

X: [-10; 10]  |  Y: [-5; 5]

🎨 Слои

📤 Сохранить

📚 Теория под вашу формулу

🔹 Область определения

D(f) = ℝ
Для 1/x: x ≠ 0
Для √x: x ≥ 0
Для log(x): x > 0

🔹 Область значений

E(f) = ℝ
sin(x): [-1; 1]
x²: [0; +∞)
eˣ: (0; +∞)

🔹 Свойства

Чётная: f(-x)=f(x) → симметрия Oy
Нечётная: f(-x)=-f(x) → симметрия начала
Период: sin/cos: 2π, tan: π

🔹 Асимптоты

Вертикальная: x=a, если f→±∞
Горизонтальная: y=b, если f→b при x→±∞
Наклонная: y=kx+m, k=lim f(x)/x

👋 Привет! Быстрый старт

Здесь вы строите графики без программирования.

Что умеет:

  • Любые формулы: 2*x+1, sin(x)*x
  • Зум колёсиком или кнопками
  • Несколько графиков одновременно
  • Теория меняется под вашу функцию

🧮 Шаг 1: Введите формулу

Напишите в поле: 3*sin(x)

Или кликните на пример выше → формула подставится.

Нажмите «Построить» или Enter.

🔍 Шаг 2: Исследуйте

• Колёсико — зум

• Перетаскивание — сдвиг

• «+ Слой» — добавить второй график

Попробуйте добавить cos(x) поверх sin(x)!
