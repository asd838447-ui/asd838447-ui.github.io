📈 Билет 21. Формулы для построения графиков функций

📊 Интерактивный график функции

1. Преобразования графиков

Перенос (сдвиг)

y = f(x) + a  →  сдвиг вдоль Oy на a
y = f(x - a)  →  сдвиг вдоль Ox на a

Растяжение/сжатие

y = k f(x)  →  растяжение вдоль Oy в k раз при k>1, сжатие при 0<k<1
y = f(kx)  →  сжатие вдоль Ox в k раз при k>1, растяжение при 0<k<1

Отражение (симметрия)

y = -f(x)  →  симметрия относительно Ox
y = f(-x)  →  симметрия относительно Oy
y = -f(-x)  →  симметрия относительно начала координат

Модуль

y = |f(x)|  →  часть графика при f(x) ≥ 0 остаётся, при f(x) < 0 отражается вверх относительно Ox
y = f(|x|)  →  график при x ≥ 0 остаётся, при x < 0 строится симметрично относительно Oy

2. Линейная функция

y = kx + b
k = tg α  — угловой коэффициент
b  — точка пересечения с Oy (0; b)

3. Квадратичная функция

y = ax² + bx + c, a ≠ 0
x₀ = -b/(2a)  — вершина (абсцисса)
y₀ = c - b²/(4a)  — вершина (ордината)
D = b² - 4ac
x₁,₂ = (-b ± √D)/(2a)  — корни при D ≥ 0
y = a(x - x₀)² + y₀  — канонический вид

4. Степенная функция

y = xⁿ, n ∈ Z
y = x^{2k}  — чётная, возрастает при x>0, убывает при x<0
y = x^{2k+1}  — нечётная, возрастает на R
y = x^{-2k}  — чётная, убывает при x>0, возрастает при x<0
y = x^{-(2k+1)}  — нечётная, убывает на (-∞;0) ∪ (0;+∞)
y = x^{1/n} = ⁿ√x  — обратная к y = xⁿ

5. Показательная функция

y = aˣ, a > 0, a ≠ 1
y = eˣ  — натуральная
При a > 1: возрастает, lim_{x→-∞} aˣ = 0, lim_{x→+∞} aˣ = +∞
При 0 < a < 1: убывает, lim_{x→-∞} aˣ = +∞, lim_{x→+∞} aˣ = 0
Горизонтальная асимптота: y = 0
Пересечение с Oy: (0; 1)

6. Логарифмическая функция

y = log_a x, a > 0, a ≠ 1
y = ln x  — натуральный
y = lg x  — десятичный
При a > 1: возрастает, lim_{x→0⁺} log_a x = -∞, lim_{x→+∞} log_a x = +∞
При 0 < a < 1: убывает, lim_{x→0⁺} log_a x = +∞, lim_{x→+∞} log_a x = -∞
Вертикальная асимптота: x = 0
Пересечение с Ox: (1; 0)

7. Тригонометрические функции

Синус

y = sin x
Область определения: R
Область значений: [-1; 1]
Период: T = 2π
Нули: x = πn, n ∈ Z
Максимумы: x = π/2 + 2πn, y = 1
Минимумы: x = -π/2 + 2πn, y = -1
Нечётная: sin(-x) = -sin x

Косинус

y = cos x
Область определения: R
Область значений: [-1; 1]
Период: T = 2π
Нули: x = π/2 + πn, n ∈ Z
Максимумы: x = 2πn, y = 1
Минимумы: x = π + 2πn, y = -1
Чётная: cos(-x) = cos x

Тангенс

y = tan x
Область определения: x ≠ π/2 + πn, n ∈ Z
Область значений: R
Период: T = π
Нули: x = πn
Вертикальные асимптоты: x = π/2 + πn
Нечётная: tan(-x) = -tan x

Котангенс

y = cot x
Область определения: x ≠ πn, n ∈ Z
Область значений: R
Период: T = π
Нули: x = π/2 + πn
Вертикальные асимптоты: x = πn
Нечётная: cot(-x) = -cot x

8. Обратные тригонометрические функции

Арксинус

y = arcsin x
Область определения: [-1; 1]
Область значений: [-π/2; π/2]
Нечётная: arcsin(-x) = -arcsin x

Арккосинус

y = arccos x
Область определения: [-1; 1]
Область значений: [0; π]
arccos(-x) = π - arccos x

Арктангенс

y = arctan x
Область определения: R
Область значений: (-π/2; π/2)
Горизонтальные асимптоты: y = π/2 (x → +∞), y = -π/2 (x → -∞)
Нечётная: arctan(-x) = -arctan x

Арккотангенс

y = arccot x
Область определения: R
Область значений: (0; π)
Горизонтальные асимптоты: y = π (x → -∞), y = 0 (x → +∞)
arccot(-x) = π - arccot x

9. Гиперболические функции

Гиперболический синус

y = sh x = (eˣ - e⁻ˣ)/2
Область определения: R
Область значений: R
Нечётная: sh(-x) = -sh x

Гиперболический косинус

y = ch x = (eˣ + e⁻ˣ)/2
Область определения: R
Область значений: [1; +∞)
Чётная: ch(-x) = ch x

Гиперболический тангенс

y = th x = sh x / ch x = (eˣ - e⁻ˣ)/(eˣ + e⁻ˣ)
Область определения: R
Область значений: (-1; 1)
Горизонтальные асимптоты: y = 1 (x → +∞), y = -1 (x → -∞)
Нечётная: th(-x) = -th x

10. Асимптоты

Вертикальные

lim_{x→a⁺} f(x) = ±∞ или lim_{x→a⁻} f(x) = ±∞ → прямая x = a

Горизонтальные

lim_{x→+∞} f(x) = b → y = b
lim_{x→-∞} f(x) = b → y = b

Наклонные

k = lim_{x→±∞} f(x)/x, b = lim_{x→±∞} (f(x) - kx) → прямая y = kx + b

11. Чётность и нечётность

Чётная: f(-x) = f(x) — симметрия относительно Oy
Нечётная: f(-x) = -f(x) — симметрия относительно начала координат

12. Периодичность

f(x + T) = f(x), T > 0 — наименьший положительный период
sin x, cos x, sec x, csc x: T = 2π
tan x, cot x: T = π
