📈 Билет 21. Формулы для построения графиков функций

🔹 Преобразования графиков

y = f(x) + a  →  сдвиг вдоль Oy на a
y = f(x − a)  →  сдвиг вдоль Ox на a
y = k·f(x)  →  растяжение вдоль Oy в k раз (k>1), сжатие (0<k<1)
y = f(kx)  →  сжатие вдоль Ox в k раз (k>1), растяжение (0<k<1)
y = −f(x)  →  симметрия относительно Ox
y = f(−x)  →  симметрия относительно Oy
y = −f(−x)  →  симметрия относительно начала координат
y = |f(x)|  →  часть графика при f(x) ≥ 0 остаётся, при f(x) < 0 отражается вверх
y = f(|x|)  →  график при x ≥ 0 остаётся, при x < 0 строится симметрично относительно Oy

🔹 Основные элементарные функции

Линейная: y = kx + b — прямая, k = tgα, b — пересечение с Oy
Квадратичная: y = ax² + bx + c, a≠0. Вершина: x₀ = −b/(2a), y₀ = c − b²/(4a)
Степенная: y = xⁿ. Чётная при чётном n, нечётная при нечётном n
Показательная: y = aˣ (a>0, a≠1). При a>1 — возрастает, при 0<a<1 — убывает. Горизонтальная асимптота y=0
Логарифмическая: y = log_a x (a>0, a≠1). При a>1 — возрастает, при 0<a<1 — убывает. Вертикальная асимптота x=0

🔹 Тригонометрические функции

sin x: D=R, E=[−1;1], T=2π, нечётная
cos x: D=R, E=[−1;1], T=2π, чётная
tan x: D≠π/2+πn, E=R, T=π, нечётная, верт. асимптоты x=π/2+πn
cot x: D≠πn, E=R, T=π, нечётная, верт. асимптоты x=πn

🔹 Обратные тригонометрические функции

arcsin x: D=[−1;1], E=[−π/2;π/2], нечётная
arccos x: D=[−1;1], E=[0;π]
arctan x: D=R, E=(−π/2;π/2), нечётная, горизонтальные асимптоты y=±π/2
arccot x: D=R, E=(0;π)

🔹 Гиперболические функции

sh x = (eˣ − e⁻ˣ)/2, D=R, E=R, нечётная
ch x = (eˣ + e⁻ˣ)/2, D=R, E=[1;+∞), чётная
th x = sh x / ch x = (eˣ − e⁻ˣ)/(eˣ + e⁻ˣ), D=R, E=(−1;1), нечётная, горизонтальные асимптоты y=±1

🔹 Асимптоты

Вертикальные: x = a, если lim_{x→a±} f(x) = ±∞
Горизонтальные: y = b, если lim_{x→±∞} f(x) = b
Наклонные: y = kx + b, где k = lim_{x→±∞} f(x)/x, b = lim_{x→±∞} (f(x) − kx)

🔹 Чётность и нечётность

Чётная: f(−x) = f(x) — симметрия относительно Oy
Нечётная: f(−x) = −f(x) — симметрия относительно начала координат

🔹 Периодичность

f(x + T) = f(x), T > 0 — наименьший положительный период
sin x, cos x, sec x, csc x: T = 2π
tan x, cot x: T = π
