📈 Билет 21. Формулы для построения графиков функций

✨ Интерактивные графики — изменяйте параметры и наблюдайте, как меняется функция в реальном времени

📐 Линейная функция

y = k·x + b прямая линия

🎛️ Параметры

📌 k = tg α — угловой коэффициент (тангенс угла наклона)
📌 b — точка пересечения с Oy: (0; b)
📌 При k > 0 — функция возрастает, при k < 0 — убывает

🔄 Квадратичная функция

y = a·x² + b·x + c парабола

🎛️ Параметры

📌 Вершина: x₀ = -b/(2a), y₀ = c - b²/(4a)
📌 Дискриминант: D = b² - 4ac
📌 Корни: x₁,₂ = (-b ± √D)/(2a)
📌 Канонический вид: y = a(x - x₀)² + y₀

📈 Показательная функция

y = aˣ экспоненциальный рост/убывание

🎛️ Параметры

📌 При a > 1: возрастает, lim_{x→-∞} aˣ = 0, lim_{x→+∞} aˣ = +∞
📌 При 0 < a < 1: убывает, lim_{x→-∞} aˣ = +∞, lim_{x→+∞} aˣ = 0
📌 Горизонтальная асимптота: y = 0
📌 Пересечение с Oy: (0; 1)
⭐ Натуральная экспонента: y = eˣ (e ≈ 2.718)

📊 Логарифмическая функция

y = log_a x обратная к показательной

🎛️ Параметры

📌 При a > 1: возрастает, lim_{x→0⁺} log_a x = -∞, lim_{x→+∞} log_a x = +∞
📌 При 0 < a < 1: убывает, lim_{x→0⁺} log_a x = +∞, lim_{x→+∞} log_a x = -∞
📌 Вертикальная асимптота: x = 0
📌 Пересечение с Ox: (1; 0)
⭐ Натуральный логарифм: y = ln x, Десятичный: y = lg x

🌀 Тригонометрические функции

y = sin x синусоида

🎛️ Параметры

📌 Область определения: R
📌 Область значений: [-1; 1]
📌 Период: T = 2π
📌 Нечётная: sin(-x) = -sin x

y = cos x косинусоида

🎛️ Параметры

📌 Область определения: R
📌 Область значений: [-1; 1]
📌 Период: T = 2π
📌 Чётная: cos(-x) = cos x

y = tg x тангенсоида

Вертикальные асимптоты при x = π/2 + πn

📌 Область определения: x ≠ π/2 + πn, n ∈ Z
📌 Область значений: R
📌 Период: T = π
📌 Нечётная: tg(-x) = -tg x
📌 Вертикальные асимптоты: x = π/2 + πn

🔄 Обратные тригонометрические функции

y = arcsin x арксинус

📌 D = [-1; 1], E = [-π/2; π/2]
Нечётная: arcsin(-x) = -arcsin x

y = arccos x арккосинус

📌 D = [-1; 1], E = [0; π]
arccos(-x) = π - arccos x

y = arctan x арктангенс

📌 D = R, E = (-π/2; π/2)
Горизонтальные асимптоты: y = π/2 (x→+∞), y = -π/2 (x→-∞)
Нечётная: arctan(-x) = -arctan x

🌀 Гиперболические функции

y = sh x = (eˣ - e⁻ˣ)/2 гиперболический синус

📌 D = R, E = R, нечётная

y = ch x = (eˣ + e⁻ˣ)/2 гиперболический косинус (цепная линия)

📌 D = R, E = [1; +∞), чётная

📐 Асимптоты, чётность и периодичность

🔹 Вертикальные асимптоты

📌 x = a, если lim_{x→a⁺} f(x) = ±∞ или lim_{x→a⁻} f(x) = ±∞

🔹 Горизонтальные асимптоты

📌 y = b, если lim_{x→±∞} f(x) = b

🔹 Наклонные асимптоты

📌 y = kx + b, где k = lim_{x→±∞} f(x)/x, b = lim_{x→±∞} (f(x) - kx)

🔹 Чётность и нечётность

📌 Чётная: f(-x) = f(x) — симметрия относительно Oy
📌 Нечётная: f(-x) = -f(x) — симметрия относительно начала координат

🔹 Периодичность

📌 f(x + T) = f(x), T > 0 — наименьший положительный период
📌 sin x, cos x: T = 2π  |  tan x, cot x: T = π
