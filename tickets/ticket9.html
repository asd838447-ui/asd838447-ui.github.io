Билет 9. Критерий Коши существования предела последовательности. Примеры (расходимость суммы \(\{1+\tfrac{1}{2}+\dots+\tfrac{1}{n}\}\), пример с доказательством сходимости по критерию Коши).

Критерий Коши — фундаментальное утверждение, позволяющее установить факт сходимости последовательности, основываясь исключительно на свойствах самой последовательности, не зная заранее значения её предела.

1. Определение фундаментальной последовательности

Определение. Последовательность \( \{x_n\} \) называется фундаментальной (последовательностью Коши), если для любого \( \varepsilon > 0 \) существует номер \( N_\varepsilon \), такой что для любых \( n, m \geq N_\varepsilon \) расстояние между членами меньше \( \varepsilon \).

\[ \forall \varepsilon > 0 \; \exists N_\varepsilon \in \mathbb{N} : \forall n, m \geq N_\varepsilon \Rightarrow |x_n - x_m| < \varepsilon \]

Эквивалентная формулировка (через «хвосты»):

\[ \forall \varepsilon > 0 \; \exists N_\varepsilon \in \mathbb{N} : \forall n \geq N_\varepsilon,\; \forall p \in \mathbb{N} \Rightarrow |x_{n+p} - x_n| < \varepsilon \]

Это означает: начиная с некоторого номера все последующие члены становятся сколь угодно близкими друг к другу.

2. Формулировка критерия Коши

Теорема. Числовая последовательность \( \{x_n\} \) имеет конечный предел (сходится) тогда и только тогда, когда она является фундаментальной.

3. Доказательство критерия Коши

Часть 1: Необходимость (Сходимость ⇒ Фундаментальность)

  1. Пусть \( \lim x_n = a \). По определению предела, для \( \varepsilon/2 > 0 \) найдётся \( N_\varepsilon \), такой что для всех \( n \geq N_\varepsilon \): \( |x_n - a| < \varepsilon/2 \).
  2. Возьмём любые \( n, m \geq N_\varepsilon \). По неравенству треугольника: \[ |x_n - x_m| = |(x_n - a) + (a - x_m)| \leq |x_n - a| + |a - x_m| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon. \]
  3. Последовательность фундаментальна. ∎

Часть 2: Достаточность (Фундаментальность ⇒ Сходимость)

Пусть \( \{x_n\} \) фундаментальна. Докажем сходимость в три этапа.

Этап 1: Ограниченность.

  1. Положим \( \varepsilon = 1 \). Найдётся \( N_1 \), такой что для всех \( n, m \geq N_1 \): \( |x_n - x_m| < 1 \).
  2. Зафиксируем \( m = N_1 \). Тогда для всех \( n \geq N_1 \): \( x_{N_1} - 1 < x_n < x_{N_1} + 1 \). «Хвост» ограничен.
  3. Первые \( N_1 - 1 \) членов ограничены как конечное множество. Вся последовательность ограничена.

Этап 2: Выделение сходящейся подпоследовательности.

  1. По теореме Больцано-Вейерштрасса из ограниченной последовательности можно выделить сходящуюся подпоследовательность \( \{x_{n_k}\} \).
  2. Пусть \( \lim_{k\to\infty} x_{n_k} = a \).

Этап 3: Сходимость всей последовательности к \( a \).

  1. Зафиксируем \( \varepsilon > 0 \). Из фундаментальности для \( \varepsilon/2 \) найдётся \( N_1 \): для всех \( n, m \geq N_1 \) выполняется \( |x_n - x_m| < \varepsilon/2 \).
  2. Из \( x_{n_k} \to a \) для \( \varepsilon/2 \) найдётся \( K \): для всех \( k \geq K \) выполняется \( |x_{n_k} - a| < \varepsilon/2 \).
  3. Выберем \( k \) так, чтобы \( k \geq K \) и \( n_k \geq N_1 \). Тогда для всех \( n \geq N_1 \): \[ |x_n - a| \leq |x_n - x_{n_k}| + |x_{n_k} - a| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon. \]
  4. По определению предела: \( \lim x_n = a \). ∎

4. Пример 1: Расходимость гармонической суммы

Рассмотрим последовательность частичных сумм гармонического ряда:

\[ x_n = 1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n} \]

Докажем расходимость, показав, что последовательность не фундаментальна. По отрицанию критерия Коши, нужно найти \( \varepsilon_0 > 0 \), такое что для любого \( N \) найдутся \( n > N \) и \( p \geq 1 \), при которых \( |x_{n+p} - x_n| \geq \varepsilon_0 \).

Рассмотрим разность \( |x_{2n} - x_n| \):

\[ |x_{2n} - x_n| = \frac{1}{n+1} + \frac{1}{n+2} + \cdots + \frac{1}{2n} \]

В этой сумме \( n \) слагаемых, и каждое из них не меньше наименьшего — последнего:

\[ \frac{1}{n+1} \geq \frac{1}{2n},\quad \frac{1}{n+2} \geq \frac{1}{2n},\quad \ldots,\quad \frac{1}{2n} = \frac{1}{2n} \]

Следовательно:

\[ |x_{2n} - x_n| \geq n \cdot \frac{1}{2n} = \frac{1}{2} \]

Нашли \( \varepsilon_0 = 1/2 \). Для любого \( N \) можно взять \( n = N \), \( p = n \) — условие фундаментальности нарушено. Последовательность расходится к \( +\infty \).

5. Пример 2: Доказательство сходимости последовательности \( x_n = 1/n \)

Возьмём произвольное \( \varepsilon > 0 \). Нужно найти \( N_\varepsilon \), такой что для всех \( n, m \geq N_\varepsilon \) выполняется \( |1/n - 1/m| < \varepsilon \).

Оценим разность:

\[ |x_n - x_m| = \left|\frac{1}{n} - \frac{1}{m}\right| \leq \frac{1}{n} + \frac{1}{m} \]

Если выбрать \( N_\varepsilon > 2/\varepsilon \), то для всех \( n, m \geq N_\varepsilon \):

\[ \frac{1}{n} \leq \frac{1}{N_\varepsilon} < \frac{\varepsilon}{2} \quad\text{и}\quad \frac{1}{m} \leq \frac{1}{N_\varepsilon} < \frac{\varepsilon}{2} \]

Тогда:

\[ |x_n - x_m| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon \]

Для любого \( \varepsilon \) нашли \( N_\varepsilon = \lfloor 2/\varepsilon \rfloor + 1 \). Последовательность фундаментальна и, по критерию Коши, сходится. ∎
