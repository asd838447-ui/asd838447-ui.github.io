Билет 9. Критерий Коши существования предела последовательности. Примеры (расходимость суммы {1+1/2+…+1/n}, пример с доказательством сходимости последовательности по критерию Коши).

Критерий Коши — это фундаментальное утверждение математического анализа, которое позволяет установить факт сходимости числовой последовательности, основываясь исключительно на свойствах самой последовательности (внутреннее свойство), не зная заранее значения её предела.

1. Определение фундаментальной последовательности

Прежде чем формулировать сам критерий, необходимо ввести понятие фундаментальной последовательности (также называемой последовательностью Коши).

Определение. Последовательность {xₙ} называется фундаментальной, если для любого сколь угодно малого положительного числа ε существует такой номер Nₑ, что для любых двух членов последовательности с номерами n и m, большими или равными Nₑ, расстояние между ними меньше ε.

Формально на языке кванторов:

∀ε > 0 ∃ Nₑ ∈ N : ∀ n, m ≥ Nₑ ⇒ |xₙ − xₘ| < ε

Существует эквивалентная формулировка, часто используемая в доказательствах:

∀ε > 0 ∃ Nₑ ∈ N : ∀ n ≥ Nₑ, ∀ p ∈ N ⇒ |xₙ₊ₚ − xₙ| < ε

Это означает, что начиная с некоторого номера все последующие члены последовательности становятся сколь угодно близкими друг к другу.

2. Формулировка критерия Коши

Теорема. Числовая последовательность {xₙ} имеет конечный предел (сходится) тогда и только тогда, когда она является фундаментальной.

3. Доказательство критерия Коши

Доказательство состоит из двух частей: необходимости и достаточности.

Часть 1: Необходимость (Сходимость ⇒ Фундаментальность)

  1. Пусть последовательность {xₙ} сходится и её предел равен a (lim xₙ = a).
  2. По определению предела, для любого ε > 0 (в частности, для ε/2) найдется такой номер Nₑ, что для всех n ≥ Nₑ выполняется неравенство: |xₙ − a| < ε/2.
  3. Возьмем любые два номера n ≥ Nₑ и m ≥ Nₑ. Оценим разность между членами xₙ и xₘ, используя неравенство треугольника (прибавим и вычтем a под знаком модуля): |xₙ − xₘ| = |(xₙ − a) + (a − xₘ)| ≤ |xₙ − a| + |a − xₘ|.
  4. Так как оба номера n и m больше Nₑ, то каждое из слагаемых меньше ε/2: |xₙ − xₘ| < ε/2 + ε/2 = ε.
  5. Таким образом, последовательность является фундаментальной.

Часть 2: Достаточность (Фундаментальность ⇒ Сходимость)

Пусть последовательность {xₙ} фундаментальна. Докажем, что она сходится, в три этапа.

Этап 1: Ограниченность фундаментальной последовательности.

  1. Положим ε = 1. Тогда существует номер N₁, такой что для всех n, m ≥ N₁ верно |xₙ − xₘ| < 1.
  2. Зафиксируем m = N₁. Тогда для всех n ≥ N₁ выполняется |xₙ − x_{N₁}| < 1, что означает x_{N₁} − 1 < xₙ < x_{N₁} + 1. Таким образом, «хвост» последовательности ограничен.
  3. Первые N₁ − 1 членов также ограничены как конечное множество. Следовательно, вся фундаментальная последовательность ограничена.

Этап 2: Выделение сходящейся подпоследовательности.

  1. Согласно теореме Больцано-Вейерштрасса, из любой ограниченной последовательности можно выделить сходящуюся подпоследовательность {xₙₖ}.
  2. Пусть она сходится к некоторому числу a: lim xₙₖ = a.

Этап 3: Сходимость всей последовательности к тому же пределу.

Нам нужно показать, что вся последовательность {xₙ} стремится к a.

  1. Зафиксируем ε > 0. Так как {xₙ} фундаментальна, для ε/2 существует номер N₁, такой что для всех n, m ≥ N₁ выполняется |xₙ − xₘ| < ε/2.
  2. Так как подпоследовательность {xₙₖ} стремится к a, для того же ε/2 найдется такой номер K, что для всех k ≥ K выполняется |xₙₖ − a| < ε/2.
  3. Выберем такое k, чтобы одновременно k ≥ K и индекс nₖ ≥ N₁ (это всегда возможно, так как nₖ → ∞).
  4. Тогда для любого n ≥ N₁ имеем: |xₙ − a| = |(xₙ − xₙₖ) + (xₙₖ − a)| ≤ |xₙ − xₙₖ| + |xₙₖ − a| < ε/2 + ε/2 = ε.
  5. Это доказывает, что lim xₙ = a. Критерий полностью доказан.

4. Пример 1: Расходимость гармонической суммы

Рассмотрим последовательность частичных сумм гармонического ряда:

xₙ = 1 + 1/2 + 1/3 + … + 1/n

Докажем её расходимость, показав, что она не является фундаментальной.

Согласно отрицанию критерия Коши, последовательность расходится, если существует такое ε₀ > 0, что для любого сколь угодно большого N можно найти такие n > N и p ≥ 1, что |xₙ₊ₚ − xₙ| ≥ ε₀.

Рассмотрим разность |x₂ₙ − xₙ|:

|x₂ₙ − xₙ| = (1 + 1/2 + … + 1/(2n)) − (1 + 1/2 + … + 1/n) = 1/(n+1) + 1/(n+2) + … + 1/(2n)

В этой сумме n слагаемых. Каждое из них не меньше самого маленького — последнего (1/(2n)):

1/(n+1) ≥ 1/(2n), 1/(n+2) ≥ 1/(2n), …, 1/(2n) = 1/(2n)

Следовательно:

|x₂ₙ − xₙ| ≥ n · (1/(2n)) = 1/2

Мы нашли ε₀ = 1/2. Для любого N мы можем взять n = N и p = n, тогда разность между членами будет не меньше 1/2. Условие фундаментальности нарушено, значит, последовательность расходится к +∞.

5. Пример 2: Доказательство сходимости последовательности xₙ = 1/n

Докажем сходимость этой последовательности, используя критерий Коши.

Возьмем произвольное ε > 0. Нам нужно найти такое Nₑ, чтобы для всех n, m ≥ Nₑ выполнялось |1/n − 1/m| < ε.

Оценим разность:

|xₙ − xₘ| = |1/n − 1/m| ≤ |1/n| + |1/m| = 1/n + 1/m

Если мы выберем Nₑ > 2/ε, то для всех n, m ≥ Nₑ будем иметь:

1/n ≤ 1/Nₑ < ε/2 и 1/m ≤ 1/Nₑ < ε/2

Тогда:

|xₙ − xₘ| < ε/2 + ε/2 = ε

Поскольку для любого ε мы смогли указать соответствующий номер Nₑ (например, Nₑ = ⌊2/ε⌋ + 1), последовательность является фундаментальной и, по критерию Коши, сходится.
