Билет 15. Односторонние пределы функции, примеры. Монотонные функции. Теорема об односторонних пределах монотонной функции и её следствие.

1. Определение односторонних пределов функции и примеры

Понятие предела функции в точке \(x_0\) характеризует поведение функции вблизи этой точки. Однако часто функция ведет себя по-разному в зависимости от того, с какой стороны (слева или справа) аргумент \(x\) приближается к \(x_0\). Для таких случаев вводятся односторонние пределы.

Определения окрестностей:

  • Левая окрестность точки \(a\): интервал \(U^-(a, \delta) = (a - \delta, a]\).
  • Правая окрестность точки \(a\): интервал \(U^+(a, \delta) = [a, a + \delta)\).
  • Проколотые односторонние окрестности (не включающие саму точку \(a\)):
    • \(\dot{U}^-(a, \delta) = (a - \delta, a)\)
    • \(\dot{U}^+(a, \delta) = (a, a + \delta)\)

Определение левого предела (по Коши):

Число \(b\) называется левым пределом функции \(f(x)\) при \(x\), стремящемся к \(a\) (обозначается \(f(a - 0)\) или \(\lim_{x \to a-0} f(x)\)), если для любого сколь угодно малого \(\varepsilon > 0\) найдется такое \(\delta > 0\), что для всех \(x\) из проколотой левой окрестности \(\dot{U}^-(a, \delta) \cap D\) выполняется неравенство \(|f(x) - b| < \varepsilon\).

Определение правого предела (по Коши):

Число \(d\) называется правым пределом функции \(f(x)\) при \(x\), стремящемся к \(a\) (обозначается \(f(a + 0)\) или \(\lim_{x \to a+0} f(x)\)), если для любого \(\varepsilon > 0\) найдется \(\delta > 0\), такое что для всех \(x\) из проколотой правой окрестности \(\dot{U}^+(a, \delta) \cap D\) выполняется \(|f(x) - d| < \varepsilon\).

Важное утверждение о связи с общим пределом:

Предел функции lim_{x→a} f(x) = b существует тогда и только тогда, когда существуют оба односторонних предела, и они равны между собой: f(a - 0) = f(a + 0) = b.

Примеры:

  • Функция скачка: Пусть f(x) = 0 при x < 0 и f(x) = 1 при x ≥ 0. Здесь левый предел в нуле f(0 - 0) = 0 (так как слева от нуля функция всегда 0), а правый предел f(0 + 0) = 1 (так как справа она 1). Поскольку 0 ≠ 1, общий предел в точке 0 не существует.
  • Функция знака (sgn(x)): В точке 0 левый предел равен −1, а правый равен 1.

2. Монотонные функции

Монотонность описывает характер изменения значений функции при изменении аргумента.

Определение: Пусть функция \(f\) задана на множестве \(E\).

  • \(f\) называется возрастающей (неубывающей), если для любых \(x_1, x_2 \in E\) из \(x_1 < x_2\) следует \(f(x_1) \le f(x_2)\). Обозначение: \(f \uparrow\).
  • \(f\) называется строго возрастающей, если из \(x_1 < x_2\) следует \(f(x_1) < f(x_2)\). Обозначение: \(f \uparrow\uparrow\).
  • \(f\) называется убывающей (невозрастающей), если из \(x_1 < x_2\) следует \(f(x_1) \ge f(x_2)\). Обозначение: \(f \downarrow\).
  • \(f\) называется строго убывающей, если из \(x_1 < x_2\) следует \(f(x_1) > f(x_2)\). Обозначение: \(f \downarrow\downarrow\).

3. Теорема об односторонних пределах монотонной функции

Эта теорема гарантирует, что монотонная функция "не может вести себя слишком плохо" — у неё всегда есть предел с каждой стороны.

Теорема: Пусть функция \(f\) монотонна на интервале \((a, b)\). Тогда:

  • В каждой точке \(c \in (a, b)\) существуют конечные левый и правый пределы \(f(c - 0)\) и \(f(c + 0)\).
  • На концах интервала существуют пределы (конечные или бесконечные) \(f(a + 0)\) и \(f(b - 0)\).

Доказательство (для случая возрастающей функции \(f \uparrow\)):

Рассмотрим точку \(c \in (a, b)\).

  1. Рассмотрим множество значений функции слева от точки \(c\), то есть на интервале \((a, c)\). Так как функция не убывает (\(f \uparrow\)), то для любого \(x \in (a, c)\) выполняется \(f(x) \le f(c)\).
  2. Это означает, что множество значений функции \(\{f(x) : x \in (a, c)\}\) ограничено сверху числом \(f(c)\). По аксиоме полноты действительных чисел, у этого множества существует точная верхняя грань (супремум). Обозначим её \(s = \sup_{x \in (a, c)} f(x)\).
  3. Покажем, что этот супремум \(s\) и есть левый предел \(f(c - 0)\). По определению супремума, для любого \(\varepsilon > 0\) найдется такое значение \(x_\varepsilon \in (a, c)\), что \(f(x_\varepsilon) > s - \varepsilon\).
  4. Так как функция \(f\) возрастает, то для любого \(x\) из интервала \((x_\varepsilon, c)\) (который является левой проколотой окрестностью точки \(c\)) будет верно неравенство: \(f(x_\varepsilon) \le f(x) \le s\).
  5. Объединяя шаги 3 и 4, получаем: \(s - \varepsilon < f(x) \le s\) для всех \(x \in (x_\varepsilon, c)\). Это означает, что \(|f(x) - s| < \varepsilon\).
  6. Следовательно, по определению предела, \(f(c - 0) = s = \sup_{x \in (a, c)} f(x)\). Конечный предел существует.

Аналогично доказывается существование правого предела через инфимум: \(f(c + 0) = \inf_{x \in (c, b)} f(x)\).

Для концов интервала (например, \(b\)) доказательство аналогично: если функция ограничена сверху на \((a, b)\), предел конечен и равен супремуму; если не ограничена — предел равен \(+\infty\).

4. Следствие о точках разрыва монотонной функции

Следствие: Если функция f монотонна на отрезке [a, b], то она может иметь точки разрыва только первого рода (скачки), и множество таких точек не более чем счетно.

Объяснение:

  1. Поскольку в каждой внутренней точке существуют конечные односторонние пределы f(c - 0) и f(c + 0), то разрыв может возникнуть только если f(c - 0) ≠ f(c + 0). Это по определению разрыв первого рода.
  2. Величина f(c + 0) - f(c - 0) называется скачком функции в этой точке.
  3. Из монотонности следует, что каждому такому скачку можно сопоставить интервал (f(c - 0), f(c + 0)). Эти интервалы для разных точек разрыва не будут пересекаться между собой.
  4. В каждом таком непустом интервале можно выбрать рациональное число. Поскольку множество рациональных чисел Q счетно, то и количество таких непересекающихся интервалов (а значит, и точек разрыва) не может быть более чем счетным.
