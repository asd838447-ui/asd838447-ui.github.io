Билет 5. Предел последовательности. Свойства сходящихся последовательностей. Лемма о знаке, теоремы о переходе к пределу в неравенстве и о зажатой последовательности.

В данном билете рассматриваются основы теории пределов — фундаментального раздела математического анализа. Мы последовательно разберем определение предела, ключевые свойства сходящихся последовательностей и важные теоремы, связывающие пределы с порядковыми отношениями (неравенствами).

1. Понятие числовой последовательности и её предела

Определение последовательности: Числовая последовательность — это упорядоченный бесконечный список чисел, где каждому натуральному числу n ∈ N ставится в соответствие действительное число xₙ ∈ R. Формально последовательность можно рассматривать как функцию f: N → R.

Строгое определение предела (по Коши): Число a называется пределом последовательности {xₙ}, если для любого (даже самого малого) положительного числа ε > 0 существует такой номер Nₑ (зависящий от ε), что для всех членов последовательности с номерами n ≥ Nₑ выполняется неравенство:

|xₙ − a| < ε

Геометрический смысл: Неравенство |xₙ − a| < ε равносильно двойному неравенству a − ε < xₙ < a + ε. Это означает, что все члены последовательности, начиная с некоторого номера Nₑ, попадают в ε-окрестность точки a.

Важная эквивалентная формулировка: число a является пределом, если вне любой сколь угодно малой ε-окрестности точки a лежит лишь конечное число элементов последовательности.

2. Свойства сходящихся последовательностей

Сходящейся называется последовательность, имеющая конечный предел. Она обладает следующими фундаментальными свойствами:

А) Единственность предела

Теорема: Если последовательность имеет предел, то он только один.

Доказательство (от противного):

  1. Предположим, что последовательность {xₙ} имеет два различных предела: a₁ и a₂, причем a₁ ≠ a₂. Пусть для определенности a₁ < a₂.
  2. Выберем такое малое ε, чтобы ε-окрестности этих точек не пересекались. Для этого достаточно взять ε = |a₂ − a₁| / 2.
  3. По определению предела, почти все члены последовательности (все, кроме конечного числа) должны лежать в окрестности Uₑ(a₁).
  4. Одновременно с этим, почти все члены должны лежать в окрестности Uₑ(a₂).
  5. Поскольку окрестности не пересекаются, член последовательности не может находиться в обеих одновременно. Это противоречие: бесконечное множество элементов не может «почти целиком» находиться в двух непересекающихся зонах.
  6. Значит, предел единственен.

Б) Ограниченность

Теорема: Любая сходящаяся последовательность является ограниченной.

Доказательство:

  1. Пусть lim xₙ = a. Возьмем в определении предела конкретное значение ε = 1.
  2. Существует номер N, такой что для всех n ≥ N выполняется |xₙ − a| < 1, что означает a − 1 < xₙ < a + 1. Таким образом, «хвост» последовательности ограничен.
  3. Осталось конечное число первых элементов: {x₁, x₂, …, xₙ₋₁}. Любое конечное множество чисел всегда ограничено своим максимумом и минимумом.
  4. Объединяя эти оценки, мы можем найти общую константу C, которая будет ограничивать абсолютно все члены последовательности по модулю (|xₙ| ≤ C). Следовательно, последовательность ограничена.

Важно: обратное неверно — ограниченная последовательность (например, xₙ = (−1)ⁿ) не всегда сходится.

3. Лемма о сохранении знака

Формулировка: Если последовательность {xₙ} сходится к ненулевому пределу a ≠ 0, то, начиная с некоторого номера, все члены последовательности будут иметь тот же знак, что и число a.

Доказательство:

  1. Пусть lim xₙ = a и a > 0.
  2. Выберем в определении предела ε = a/2 (это положительное число).
  3. Тогда существует номер n₀, такой что для всех n > n₀ выполняется: |xₙ − a| < a/2 ⟹ a − a/2 < xₙ < a + a/2 ⟹ a/2 < xₙ < 3a/2.
  4. Так как a/2 > 0, то и все xₙ > 0 для всех n > n₀.
  5. Аналогично доказывается для случая a < 0 (выбирается ε = |a|/2).

4. Теорема о предельном переходе в неравенстве

Формулировка: Если последовательности {xₙ} и {yₙ} сходятся к пределам a и b соответственно, и начиная с некоторого момента xₙ ≤ yₙ, то и для их пределов выполняется неравенство a ≤ b.

Доказательство (от противного):

  1. Предположим обратное: a > b.
  2. Тогда разность a − b > 0. Выберем достаточно малое ε = (a − b)/2.
  3. По определению предела:
    • Начиная с номера n₁, все xₙ лежат в (a−ε, a+ε). В частности, xₙ > a−ε = a − (a−b)/2 = (a+b)/2.
    • Начиная с номера n₂, все yₙ лежат в (b−ε, b+ε). В частности, yₙ < b+ε = b + (a−b)/2 = (a+b)/2.
  4. Для всех номеров n, больших максимума из n₁ и n₂, одновременно выполняются оба условия: yₙ < (a+b)/2 < xₙ.
  5. Мы получили, что yₙ < xₙ, что прямо противоречит условию xₙ ≤ yₙ.
  6. Следовательно, наше допущение было ложным и a ≤ b.

Примечание: даже если неравенство между членами было строгим (xₙ < yₙ), предел всё равно может быть нестрогим (например, 1/n > 0, но предел равен 0).

5. Теорема о зажатой последовательности (О двух милиционерах)

Формулировка: Пусть даны три последовательности {xₙ}, {yₙ} и {zₙ}. Если xₙ ≤ yₙ ≤ zₙ для всех n, начиная с некоторого номера, и крайние последовательности стремятся к одному и тому же пределу a (lim xₙ = lim zₙ = a), то средняя последовательность {yₙ} также имеет предел, равный a.

Доказательство:

  1. Зафиксируем произвольное ε > 0.
  2. Так как xₙ → a, то существует номер n₁, такой что для всех n > n₁: a − ε < xₙ < a + ε.
  3. Так как zₙ → a, то существует номер n₂, такой что для всех n > n₂: a − ε < zₙ < a + ε.
  4. Пусть n₀ — номер, начиная с которого выполняется неравенство xₙ ≤ yₙ ≤ zₙ. Возьмем N = max(n₁, n₂, n₀).
  5. Тогда для всех n > N мы можем записать цепочку неравенств: a − ε < xₙ ≤ yₙ ≤ zₙ < a + ε.
  6. Выбрасывая промежуточные части, получаем: a − ε < yₙ < a + ε.
  7. Это в точности означает, что |yₙ − a| < ε. Согласно определению предела, lim yₙ = a.

Геометрический смысл: Если две функции «зажимают» третью и обе стремятся к одной двери (точке a), то и зажатая между ними последовательность неизбежно пройдет в ту же дверь.
