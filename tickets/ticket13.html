Билет 13. Предел функции: определения по Коши и по Гейне (Коши => Гейне с доказательством). Простейшие свойства предела функции. Предел функции на бесконечности и бесконечно большие функции, примеры. Арифметические свойства предела функции.

Ниже представлен подробный ответ по билету №13.

1. Определение предела функции по Коши (на языке ε-δ)

Определение предела функции по Коши описывает поведение функции вблизи определенной точки x₀, но не обязательно в самой этой точке. Это принципиально отличает предел от значения функции в точке.

Определение. Пусть функция f(x) определена в некоторой проколотой окрестности точки x₀. Число A называется пределом функции f(x) при x, стремящемся к x₀, если для любого (сколь угодно малого) положительного числа ε > 0 существует такое положительное число δ > 0 (зависящее от ε), что для всех x из проколотой δ-окрестности точки x₀ (то есть таких x, для которых выполняется неравенство 0 < |x - x₀| < δ) значение функции f(x) попадает в ε-окрестность точки A (то есть |f(x) - A| < ε).

Символически это записывается так:

lim_{x→x₀} f(x) = A ⟺ ∀ε > 0 ∃δ(ε) > 0 : ∀x ∈ D, 0 < |x - x₀| < δ ⟹ |f(x) - A| < ε

Геометрический смысл: каким бы узким ни был горизонтальный ε-коридор вдоль прямой y = A, всегда можно найти такой вертикальный δ-коридор вокруг точки x₀, что график функции (за исключением, возможно, самой точки x₀) будет целиком лежать внутри этого горизонтального коридора.

2. Определение предела функции по Гейне (на языке последовательностей)

Определение по Гейне связывает понятие предела функции с пределами числовых последовательностей.

Определение. Число A называется пределом функции f(x) при x, стремящемся к x₀, если для любой последовательности {xₙ}, составленной из элементов области определения функции, такой, что она сходится к x₀ (xₙ → x₀ при n → ∞) и ни один её член не равен x₀ (xₙ ≠ x₀ для всех n), соответствующая последовательность значений функции {f(xₙ)} сходится к числу A.

Символически:

lim_{x→x₀} f(x) = A ⟺ ∀{xₙ}: xₙ → x₀, xₙ ≠ x₀ ⟹ f(xₙ) → A

Определения по Коши и по Гейне полностью эквивалентны.

3. Доказательство: Определение по Коши ⇒ Определение по Гейне

Дано: Предел функции f(x) в смысле Коши равен A, то есть:

∀ε > 0 ∃δ > 0 : ∀x ∈ D, 0 < |x - x₀| < δ ⟹ |f(x) - A| < ε

Нужно доказать: Для любой последовательности {xₙ}, такой что xₙ → x₀ и xₙ ≠ x₀, выполняется f(xₙ) → A.

Доказательство:

  1. Рассмотрим произвольную последовательность {xₙ}, сходящуюся к x₀, где все xₙ ≠ x₀.
  2. Зададим произвольное число ε > 0.
  3. По условию (согласно определению по Коши) для этого ε существует такое δ > 0, что как только аргумент x попадает в проколотую δ-окрестность x₀, значение f(x) попадает в ε-окрестность A.
  4. Так как последовательность {xₙ} сходится к x₀, то по определению предела последовательности для найденного нами числа δ (выступающего здесь в роли "эпсилон" для последовательности) найдется такой номер N, что для всех номеров n ≥ N выполняется неравенство |xₙ - x₀| < δ.
  5. Поскольку по условию xₙ ≠ x₀, то для всех n ≥ N выполняется условие 0 < |xₙ - x₀| < δ.
  6. Следовательно, для всех этих членов последовательности выполняется условие из определения Коши: |f(xₙ) - A| < ε.
  7. Мы показали, что для любого ε > 0 нашелся такой номер N, что при всех n ≥ N выполняется |f(xₙ) - A| < ε. Это в точности означает, что предел числовой последовательности {f(xₙ)} равен A.

Утверждение доказано.

4. Простейшие свойства предела функции

Если предел функции lim_{x→x₀} f(x) существует и равен конечному числу, то справедливы следующие свойства:

  • Единственность предела: Если функция имеет предел, то он только один.
  • Ограниченность: Если функция имеет конечный предел в точке x₀, то она ограничена в некоторой окрестности этой точки.
  • Сохранение знака (Лемма о знаке): Если lim_{x→x₀} f(x) = A ≠ 0, то существует такая окрестность точки x₀, в которой значения функции имеют тот же знак, что и число A. Более точно, если A > 0, то в этой окрестности f(x) > A/2.
  • Предел модуля: Если lim_{x→x₀} f(x) = A, то существует lim_{x→x₀} |f(x)| = |A|.
  • Предел константы: Если функция f(x) = C на некотором множестве, то её предел в любой точке прикосновения этого множества равен C.

5. Предел функции на бесконечности и бесконечно большие функции

Предел функции на бесконечности

Эти определения используются для описания поведения функции, когда аргумент неограниченно растет или убывает.

  • При x → +∞: Число b называется пределом функции, если для любого ε > 0 существует такое число M (окрестность бесконечности), что для всех x > M выполняется |f(x) - b| < ε.
  • При x → −∞: Аналогично, если для любого ε > 0 найдется M, такое что для всех x < -M выполняется |f(x) - b| < ε.

Бесконечно большие функции

Определение. Функция f(x) называется бесконечно большой при x → x₀, если её предел равен бесконечности (∞, +∞ или −∞). На языке ε-δ: для любого сколь угодно большого числа E > 0 существует δ > 0, такое что для всех x из проколотой δ-окрестности x₀ выполняется |f(x)| > E.

Примеры:

  • Функция f(x) = 1/xⁿ является бесконечно большой при x → 0.
  • Функция f(x) = xⁿ является бесконечно большой при x → ∞.
  • Показательная функция f(x) = aˣ при a > 1 является бесконечно большой при x → +∞, так как lim_{x→+∞} aˣ = +∞.
  • Функция f(x) = ln x является бесконечно большой при x → +0 (стремится к −∞).

6. Арифметические свойства предела функции

Пусть функции f(x) и g(x) имеют конечные пределы при x → x₀: lim_{x→x₀} f(x) = A и lim_{x→x₀} g(x) = B. Тогда:

  • Предел суммы (разности): Предел суммы функций равен сумме их пределов:

    lim_{x→x₀} (f(x) ± g(x)) = A ± B

  • Предел произведения: Предел произведения функций равен произведению их пределов:

    lim_{x→x₀} (f(x) · g(x)) = A · B

  • Вынесение константы: Постоянный множитель можно выносить за знак предела:

    lim_{x→x₀} (c · f(x)) = c · lim_{x→x₀} f(x) = c · A

  • Предел частного: Предел отношения функций равен отношению их пределов при условии, что предел знаменателя не равен нулю:

    lim_{x→x₀} \frac{f(x)}{g(x)} = \frac{A}{B}, \text{ при } B \neq 0

Все эти свойства напрямую доказываются с использованием определения предела по Гейне и соответствующих арифметических свойств пределов последовательностей. Достаточно рассмотреть произвольную последовательность xₙ → x₀, тогда f(xₙ) → A и g(xₙ) → B, и применить теоремы для последовательностей.
