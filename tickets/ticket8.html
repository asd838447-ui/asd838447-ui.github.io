Билет 8. Подпоследовательности, примеры. Теорема Больцано-Вейерштрасса. Утверждение о пределе подпоследовательности сходящейся последовательности.

В данном билете мы разберем понятие подпоследовательности, изучим её связь со сходящимися последовательностями и докажем одну из фундаментальных теорем математического анализа — теорему Больцано-Вейерштрасса.

1. Подпоследовательности: определение и примеры

Чтобы понять, что такое подпоследовательность, представьте основную последовательность как бесконечный пронумерованный список чисел x₁, x₂, x₃, …, xₙ, ….

Определение. Подпоследовательностью последовательности {xₙ} называется последовательность, составленная из элементов исходной последовательности {xₙ}, взятых в том же порядке, но с индексами nₖ, которые сами образуют строго возрастающую последовательность натуральных чисел: n₁ < n₂ < n₃ < … < nₖ < …

Формально подпоследовательность записывается как {xₙₖ}. Нижний индекс k указывает на порядковый номер элемента в новой подпоследовательности, а индекс nₖ показывает, какой именно номер этот элемент имел в исходной последовательности.

Примеры:

  • Последовательность обратных чисел: xₙ = 1/n = {1, 1/2, 1/3, …}. Если мы возьмем только четные индексы (nₖ = 2k), получим подпоследовательность {x₂ₖ} = {1/2, 1/4, 1/6, …}.
  • Мигающая последовательность: xₙ = (−1)ⁿ = {−1, 1, −1, 1, …}. Здесь можно выделить две важные подпоследовательности:
    • С нечетными индексами: x₂ₖ₋₁ = {−1, −1, −1, …}, которая всегда равна −1.
    • С четными индексами: x₂ₖ = {1, 1, 1, …}, которая всегда равна 1.
  • Тригонометрическая последовательность: xₙ = cos(πn/4). Выписывая её члены, мы увидим значения {√2/2, 0, −√2/2, −1, …}. Из неё можно выделить подпоследовательности, стремящиеся к разным числам (эти числа называются частичными пределами), например, подпоследовательность x₈ₖ, где все элементы равны cos(2πk) = 1, стремится к 1.

2. Утверждение о пределе подпоследовательности сходящейся последовательности

Это утверждение говорит о том, что если вся последовательность «целится» в одну точку, то любая её часть (выбранная по правилам подпоследовательности) будет «целиться» туда же.

Теорема. Если последовательность {xₙ} сходится к некоторому конечному пределу a, то любая её подпоследовательность {xₙₖ} также сходится к тому же пределу a.

Доказательство:

  1. Пусть lim xₙ = a. По определению предела это значит, что для любого ε > 0 найдется такой номер Nₑ, что для всех номеров n > Nₑ выполняется неравенство |xₙ − a| < ε.
  2. Рассмотрим произвольную подпоследовательность {xₙₖ}. По условию определения подпоследовательности, последовательность индексов {nₖ} строго возрастает (n₁ < n₂ < n₃ < …). Так как это натуральные числа, то выполняется неравенство nₖ ≥ k для любого k.
  3. Нам нужно показать, что начиная с некоторого номера k, все xₙₖ попадут в ε-окрестность точки a. Возьмем тот же самый номер Nₑ, который мы нашли для исходной последовательности.
  4. Если мы выберем k > Nₑ, то, учитывая, что nₖ ≥ k, получим nₖ > Nₑ.
  5. Поскольку индекс nₖ больше Nₑ, то для элемента xₙₖ выполняется условие из пункта 1: |xₙₖ − a| < ε.
  6. Следовательно, для любого ε > 0 мы нашли номер K = Nₑ, такой что для всех k > K выполняется |xₙₖ − a| < ε. Это и означает по определению, что lim xₙₖ = a. Ч.т.д.

3. Теорема Больцано-Вейерштрасса

Эта теорема — один из «столпов» анализа. Она гарантирует, что если последовательность не улетает в бесконечность (ограничена), то в ней обязательно можно найти «сгущение» элементов вокруг какой-то точки.

Формулировка: Из любой ограниченной последовательности можно выделить сходящуюся подпоследовательность.

Напоминание: Последовательность называется ограниченной, если все её члены по модулю не превосходят некоторого числа C (|xₙ| ≤ C для всех n), то есть лежат на отрезке [−C, C].

Доказательство (метод деления отрезка пополам):

  1. Пусть {xₙ} — ограниченная последовательность. Значит, все её члены лежат в некотором замкнутом отрезке I₀ = [a, b] (где a = −C, b = C).
  2. Разделим отрезок I₀ пополам точкой c = (a + b)/2. Получим два отрезка: [a, c] и [c, b].
  3. Так как в последовательности бесконечно много членов, то хотя бы в одном из этих двух отрезков (левом или правом) также должно содержаться бесконечно много членов последовательности {xₙ}. Обозначим этот отрезок I₁ = [a₁, b₁]. Его длина в 2 раза меньше исходного: b₁ − a₁ = (b − a)/2.
  4. Повторим процедуру: разделим I₁ пополам и выберем ту половину, где лежит бесконечно много членов {xₙ}. Назовем её I₂. Продолжая этот процесс бесконечно, мы построим систему вложенных отрезков I₀ ⊃ I₁ ⊃ I₂ ⊃ … ⊃ Iₖ ⊃ …
  5. Длина каждого следующего отрезка стремится к нулю: d(Iₖ) = (b − a)/2ᵏ → 0 при k → ∞.
  6. По теореме Кантора (о вложенных отрезках), существует единственная точка γ, принадлежащая всем этим отрезкам одновременно.
  7. Теперь сконструируем (выделим) саму подпоследовательность {xₙₖ}:
    • В качестве xₙ₁ возьмем любой элемент исходной последовательности, лежащий в I₁.
    • В качестве xₙ₂ возьмем элемент из I₂ с номером n₂ > n₁ (такой всегда найдется, так как в I₂ бесконечно много элементов).
    • На k-м шаге берем xₙₖ ∈ Iₖ так, чтобы nₖ > nₖ₋₁.
  8. Докажем, что эта подпоследовательность {xₙₖ} сходится к γ. Так как xₙₖ ∈ Iₖ и γ ∈ Iₖ, то расстояние между ними не может быть больше длины самого отрезка: |xₙₖ − γ| ≤ bₖ − aₖ.
  9. Поскольку длина отрезков bₖ − aₖ стремится к нулю, то по теореме о двух милиционерах (зажатой последовательности), |xₙₖ − γ| также стремится к нулю.
  10. Это означает, что lim xₙₖ = γ. Мы успешно выделили сходящуюся подпоследовательность из ограниченной последовательности. Ч.т.д.

Важное замечание: Если последовательность неограничена, из неё тоже можно выделить подпоследовательность, но она будет стремиться к бесконечности (+∞, −∞ или ∞), что не считается сходимостью в классическом смысле (к числу).
