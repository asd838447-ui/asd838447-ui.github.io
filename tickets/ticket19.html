Билет 19. Теорема о композиции непрерывных функций. Два замечательных предела. Примеры их использования.

Вот подробный ответ на билет №19, составленный на основе предоставленных источников в экзаменационном стиле.

1. Теорема о непрерывности сложной функции (композиции)

Определение композиции: Пусть заданы две функции: y = f(x) и z = g(y). Если область значений функции f входит в область определения функции g, то можно определить сложную функцию h(x) = g(f(x)), которую называют композицией функций.

Теорема: Если функция f(x) непрерывна в точке x₀, а функция g(y) непрерывна в точке y₀ = f(x₀), то сложная функция h(x) = g(f(x)) также непрерывна в точке x₀.

Доказательство (через определение по Гейне):

  1. Возьмем произвольную последовательность {xₙ}, сходящуюся к x₀ (xₙ → x₀).
  2. Так как функция f(x) непрерывна в точке x₀, то по определению непрерывности по Гейне последовательность соответствующих значений функции {f(xₙ)} стремится к f(x₀). Обозначим yₙ = f(xₙ) и y₀ = f(x₀). Таким образом, yₙ → y₀.
  3. Поскольку функция g(y) непрерывна в точке y₀, то для любой последовательности {yₙ}, стремящейся к y₀, последовательность значений {g(yₙ)} стремится к g(y₀).
  4. Следовательно, g(f(xₙ)) → g(f(x₀)) при n → ∞. Это в точности соответствует определению непрерывности функции h(x) = g(f(x)) в точке x₀.

Важное следствие: Данная теорема позволяет «вносить» предел внутрь знака непрерывной функции:

lim_{x→x₀} g(f(x)) = g(lim_{x→x₀} f(x)) = g(f(x₀))

Это фундаментальное свойство используется для раскрытия неопределенностей и вычисления сложных пределов.

2. Первый замечательный предел

Формулировка: Предел отношения синуса малого угла к самому углу (выраженному в радианах) при стремлении угла к нулю равен единице:

lim_{x→0} (sin x) / x = 1

Геометрическое доказательство:

  1. Рассмотрим единичную окружность (радиус R = 1). Пусть угол x положительный и малый (0 < x < π/2).
  2. На чертеже сравним площади трех фигур: треугольника OAB, сектора OAB и прямоугольного треугольника OCB. Из построения видно, что:

    S(△OAB) < S(сект.OAB) < S(△OCB)

  3. Выразим площади через угол x:
    • S(△OAB) = (1/2) · 1 · 1 · sin x = (1/2) sin x
    • S(сект.OAB) = (1/2) · 1² · x = (1/2) x
    • S(△OCB) = (1/2) · 1 · tan x = (1/2) tan x
  4. Подставим в неравенство и сократим на 1/2: sin x < x < tan x
  5. Разделим все части на sin x (так как sin x > 0 при данных условиях): 1 < x / sin x < 1 / cos x
  6. Перевернем дроби, изменив знаки неравенства: cos x < sin x / x < 1
  7. При x → 0 функция cos x → 1 (в силу непрерывности косинуса).
  8. По теореме «о двух милиционерах» (о сжатой функции), если левая и правая части стремятся к 1, то и функция посередине также стремится к 1.

Примечание: для отрицательных x доказательство аналогично в силу четности функций cos x и sin x / x.

Следствия (эквивалентности при x → 0): Из первого замечательного предела вытекают важные эквивалентности, которые используются для упрощения вычислений:

  • sin x ∼ x
  • tan x ∼ x
  • arcsin x ∼ x
  • arctan x ∼ x
  • 1 - cos x ∼ x²/2

3. Второй замечательный предел

Формулировка: Второй замечательный предел определяет число e (число Эйлера, ≈ 2.718):

  • Для последовательности (n ∈ N): lim_{n→∞} (1 + 1/n)ⁿ = e
  • Для функции (x ∈ R): lim_{x→∞} (1 + 1/x)ˣ = e
  • Форма при x → 0: lim_{x→0} (1 + x)^{1/x} = e

Идея доказательства (для функции): Доказывается с использованием свойств монотонности и ограниченности последовательностей. Показывается, что последовательность xₙ = (1 + 1/n)ⁿ возрастает и ограничена сверху, а значит, по теореме Вейерштрасса, имеет конечный предел. Затем, используя теорему о двух милиционерах, этот результат распространяется на непрерывный аргумент x → ∞.

Следствия (эквивалентности при x → 0):

  • ln(1 + x) ∼ x
  • eˣ - 1 ∼ x
  • aˣ - 1 ∼ x ln a
  • (1 + x)ᵃ - 1 ∼ a x

4. Примеры использования замечательных пределов

Пример 1 (на 1-й замечательный предел): Вычислить lim_{x→0} sin(5x) / x.

Решение: Используем эквивалентность sin(z) ∼ z при z → 0. Здесь z = 5x.

lim_{x→0} sin(5x) / x = lim_{x→0} (5x) / x = 5

Пример 2 (на эквивалентность из 1-го предела): Вычислить lim_{x→0} (1 - cos x) / x².

Решение: Используем следствие 1 - cos x ∼ x²/2.

lim_{x→0} (1 - cos x) / x² = (x²/2) / x² = 1/2

Пример 3 (на 2-й замечательный предел): Вычислить lim_{x→0} (1 + 2x)^{1/x}.

Решение: Имеем неопределенность вида [1^∞]. Приведем к виду второй формы предела lim_{t→0} (1 + t)^{1/t} = e. Пусть t = 2x. Тогда 1/x = 2/t.

lim_{x→0} (1 + 2x)^{1/x} = lim_{t→0} ((1 + t)^{1/t})² = e²

Пример 4 (комбинированный): Вычислить lim_{x→0} (e^{9x} - 1) / ln(1 + 3x).

Решение: Используем эквивалентности: e^{9x} - 1 ∼ 9x и ln(1 + 3x) ∼ 3x.

lim_{x→0} (e^{9x} - 1) / ln(1 + 3x) = lim_{x→0} (9x) / (3x) = 3
