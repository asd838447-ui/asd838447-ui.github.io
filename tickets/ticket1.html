Билет 1. Принцип математической индукции. Бином Ньютона. Неравенство Бернулли.

Для ответа на этот экзаменационный вопрос мы последовательно разберем принцип математической индукции как метод доказательства, а затем с его помощью выведем формулу бинома Ньютона и докажем неравенство Бернулли.

1. Принцип математической индукции

Принцип математической индукции — это фундаментальный метод доказательства утверждений, зависящих от натурального числа n. Суть метода заключается в том, что утверждение считается истинным для всех натуральных чисел, если выполняются два условия:

  • База индукции: Утверждение верно для начального значения (обычно для n=1).
  • Индукционный переход: Из предположения о том, что утверждение верно для некоторого произвольного числа k (предположение индукции), следует его истинность для следующего за ним числа k+1.

Если оба шага выполнены, то согласно принципу математической индукции формула или свойство доказано для любого натурального n.

2. Бином Ньютона

Формула бинома Ньютона описывает разложение произвольной натуральной степени двучлена (a+b)ⁿ в виде многочлена.

Основные понятия: факториал и биномиальные коэффициенты

Для записи формулы необходимо определить понятие факториала и биномиального коэффициента.

  • Факториал числа n (обозначается n!) — это произведение всех натуральных чисел от 1 до n: n! = 1·2·3·…·n. По определению полагают, что 0! = 1.
  • Биномиальный коэффициент (Cₙᵏ или (ⁿₖ)) определяет число способов выбрать k элементов из n и вычисляется по формуле: Cₙᵏ = \frac{n!}{k!(n-k)!}.

Формула разложения

Общий вид формулы бинома Ньютона выглядит следующим образом:

(a+b)ⁿ = ∑ₖ₌₀ⁿ Cₙᵏ aⁿ⁻ᵏ bᵏ = Cₙ⁰ aⁿ + Cₙ¹ aⁿ⁻¹ b + Cₙ² aⁿ⁻² b² + ... + Cₙⁿ bⁿ

Важной особенностью является то, что в каждом слагаемом сумма показателей степеней a и b всегда равна n. Коэффициенты разложения можно также находить с помощью треугольника Паскаля, где каждое число равно сумме двух чисел, стоящих над ним в предыдущей строке.

Доказательство методом математической индукции

Докажем формулу (a+b)ⁿ = ∑ₖ₌₀ⁿ Cₙᵏ aᵏ bⁿ⁻ᵏ.

  • База индукции (n=1): (a+b)¹ = a+b. С правой стороны: C₁⁰ a⁰ b¹ + C₁¹ a¹ b⁰ = 1·1·b + 1·a·1 = a+b. Утверждение верно.
  • Предположение индукции: Пусть формула верна для n=l, то есть: (a+b)ˡ = Cₗ⁰ a⁰ bˡ + Cₗ¹ a¹ bˡ⁻¹ + ... + Cₗˡ aˡ b⁰.
  • Индукционный переход (n=l+1): Рассмотрим (a+b)ˡ⁺¹ = (a+b)ˡ (a+b). Подставим наше предположение: (a+b)ˡ⁺¹ = (Cₗ⁰ a⁰ bˡ + Cₗ¹ a¹ bˡ⁻¹ + ... + Cₗˡ aˡ b⁰)(a+b). Раскроем скобки, умножая сначала на a, затем на b: = (Cₗ⁰ a¹ bˡ + Cₗ¹ a² bˡ⁻¹ + ... + Cₗˡ aˡ⁺¹ b⁰) + (Cₗ⁰ a⁰ bˡ⁺¹ + Cₗ¹ a¹ bˡ + ... + Cₗˡ aˡ b¹). Приведем подобные слагаемые при одинаковых степенях a и b: = Cₗ⁰ a⁰ bˡ⁺¹ + (Cₗ⁰ + Cₗ¹) a¹ bˡ + (Cₗ¹ + Cₗ²) a² bˡ⁻¹ + ... + Cₗˡ aˡ⁺¹ b⁰. Используя лемму о свойстве биномиальных коэффициентов Cₗᵏ⁻¹ + Cₗᵏ = Cₗ₊₁ᵏ, а также тот факт, что Cₗ⁰ = Cₗ₊₁⁰ = 1 и Cₗˡ = Cₗ₊₁ˡ⁺¹ = 1, получаем: (a+b)ˡ⁺¹ = Cₗ₊₁⁰ a⁰ bˡ⁺¹ + Cₗ₊₁¹ a¹ bˡ + ... + Cₗ₊₁ˡ⁺¹ aˡ⁺¹ b⁰. Формула доказана для n=l+1, а значит, по принципу индукции, она верна для любого натурального n.

3. Неравенство Бернулли

Неравенство Бернулли утверждает, что для любого действительного числа a ≥ −1 и любого натурального n справедливо:

(1 + a)ⁿ ≥ 1 + na

Доказательство методом математической индукции

  • База индукции (n=1): (1+a)¹ ≥ 1 + 1·a, что сводится к 1+a ≥ 1+a. Это истинное равенство, база доказана.
  • Предположение индукции: Пусть неравенство верно для n=k, то есть (1+a)ᵏ ≥ 1 + ka.
  • Индукционный переход (n=k+1): Рассмотрим выражение для степени k+1: (1+a)ᵏ⁺¹ = (1+a)ᵏ (1+a). Так как по условию a ≥ −1, то множитель (1+a) ≥ 0. Это позволяет нам умножить обе части неравенства из предположения индукции на (1+a), не меняя знак неравенства: (1+a)ᵏ (1+a) ≥ (1 + ka)(1 + a). Раскроем скобки в правой части: (1 + ka)(1 + a) = 1 + a + ka + ka² = 1 + (k+1)a + ka². Поскольку величина ka² всегда неотрицательна (k — натуральное, a² ≥ 0), мы можем отбросить это слагаемое, тем самым только уменьшив правую часть: 1 + (k+1)a + ka² ≥ 1 + (k+1)a. Объединяя шаги, получаем: (1+a)ᵏ⁺¹ ≥ 1 + (k+1)a. Переход доказан, следовательно, неравенство Бернулли верно для всех n∈N.
