Билет 1. Принцип математической индукции. Бином Ньютона. Неравенство Бернулли.

Ниже представлен подробный ответ по вопросам билета: принцип математической индукции, бином Ньютона и неравенство Бернулли, составленный на основе предоставленных академических источников.

1. Принцип математической индукции

Суть метода: Принцип математической индукции — это фундаментальный метод доказательства утверждений, зависящих от натурального числа n. Он базируется на аксиоматике натуральных чисел и позволяет доказать истинность утверждения для всех n∈N.

Алгоритм доказательства состоит из двух обязательных шагов:

  • База индукции: Проверяется справедливость утверждения для наименьшего возможного значения n (обычно для n=1).
  • Индукционный переход: Предполагается, что утверждение верно для некоторого произвольного натурального числа n=k (или l), и на основании этого доказывается, что оно будет верно и для следующего за ним числа n=k+1.

Если оба шага выполнены, утверждение считается доказанным для любого натурального n.

2. Бином Ньютона

Прежде чем формулировать теорему о биноме Ньютона, необходимо ввести вспомогательные понятия: факториал и биномиальные коэффициенты.

Вспомогательные определения

  • Факториал: Для натурального числа n полагают n! = 1·2·…·n. По определению принимается, что 0! = 1.
  • Биномиальные коэффициенты: Числа, обозначаемые Cₙᵏ (или (ⁿₖ)), вычисляются по формуле: $$C_n^k = \frac{n!}{k!(n - k)!}, \text{ где } n \ge k \text{ и } k \in \mathbb{N} \cup \{0\}$$. Очевидно, что для любого n выполняется Cₙ⁰ = Cₙⁿ = 1.

Лемма о сложении биномиальных коэффициентов

Для доказательства бинома Ньютона используется важное свойство:

Cₙᵏ⁻¹ + Cₙᵏ = Cₙ₊₁ᵏ

Доказательство леммы:

Cₙᵏ⁻¹ + Cₙᵏ = \frac{n!}{(k-1)!(n-k+1)!} + \frac{n!}{k!(n-k)!}

Приводя к общему знаменателю k!(n−k+1)!, получаем: $$\frac{n!k + n!(n - k + 1)}{k!(n - k + 1)!} = \frac{n!(k + n - k + 1)}{k!(n - k + 1)!} = \frac{n!(n + 1)}{k!(n + 1 - k)!} = \frac{(n + 1)!}{k!(n + 1 - k)!} = C_{n+1}^k$$

Теорема о биноме Ньютона

Для любых действительных чисел a и b и любого натурального n справедливо равенство: $$(a + b)^n = \sum_{k=0}^n C_n^k a^k b^{n-k} = b^n + C_n^1 a b^{n-1} + \dots + C_n^k a^k b^{n-k} + \dots + a^n$$

Доказательство методом индукции:

  • База: При n=1: (a+b)¹ = C₁⁰ a⁰ b¹ + C₁¹ a¹ b⁰ = 1·1·b + 1·a·1 = a+b. Утверждение верно.
  • Переход: Пусть формула верна для n=l. Докажем для n=l+1: $$(a + b)^{l+1} = (a + b)^l(a + b)$$ Подставим разложение для (a+b)ˡ: (Cₗ⁰ a⁰ bˡ + Cₗ¹ a¹ bˡ⁻¹ + … + Cₗˡ aˡ b⁰)(a+b). Раскроем скобки, умножая сначала на a, затем на b: =(Cₗ⁰ a¹ bˡ + Cₗ¹ a² bˡ⁻¹ + … + Cₗˡ aˡ⁺¹ b⁰)+(Cₗ⁰ a⁰ bˡ⁺¹ + Cₗ¹ a¹ bˡ + … + Cₗˡ aˡ b¹). Приведем подобные слагаемые при одинаковых степенях a и b: $$= C_l^0 a^0 b^{l+1} + (C_l^0 + C_l^1) a^1 b^l + \dots + (C_l^{l-1} + C_l^l) a^l b^1 + C_l^l a^{l+1} b^0$$ Применяя доказанную выше лемму (Cₗᵏ⁻¹ + Cₗᵏ = Cₗ₊₁ᵏ) и учитывая, что Cₗ⁰ = Cₗ₊₁⁰ = 1 и Cₗˡ = Cₗ₊₁ˡ⁺¹ = 1: (a+b)ˡ⁺¹ = Cₗ₊₁⁰ a⁰ bˡ⁺¹ + Cₗ₊₁¹ a¹ bˡ + … + Cₗ₊₁ˡ aˡ b¹ + Cₗ₊₁ˡ⁺¹ aˡ⁺¹ b⁰. Формула доказана для n=l+1. Следовательно, по принципу индукции, она верна для всех n∈N.

3. Неравенство Бернулли

Неравенство Бернулли используется для оценки степенных выражений и является важным инструментом математического анализа.

Формулировка: Для любого действительного числа a ≥ −1 и любого натурального числа n выполняется неравенство: $$(1 + a)^n \ge 1 + na$$

Доказательство методом индукции:

  • База: При n=1: (1+a)¹ ≥ 1+1·a, что сводится к 1+a ≥ 1+a. Утверждение верно.
  • Переход: Предположим, что неравенство верно для n=k: (1+a)ᵏ ≥ 1+ka. Докажем его справедливость для n=k+1. Умножим обе части предположения на (1+a). Так как по условию a ≥ −1, то (1+a) ≥ 0, и знак неравенства не изменится: $$(1 + a)^{k+1} = (1 + a)^k(1 + a) \ge (1 + ka)(1 + a)$$ Раскроем скобки в правой части: $$(1 + ka)(1 + a) = 1 + a + ka + ka^2 = 1 + (k + 1)a + ka^2$$ Заметим, что слагаемое ka² ≥ 0 (так как k∈N, а квадрат любого числа не отрицателен). Если мы отбросим это положительное слагаемое, сумма только уменьшится: $$1 + (k + 1)a + ka^2 \ge 1 + (k + 1)a$$ Таким образом, мы получили: $$(1 + a)^{k+1} \ge 1 + (k + 1)a$$ Переход доказан. Неравенство Бернулли справедливо для всех натуральных n и a ≥ −1.
