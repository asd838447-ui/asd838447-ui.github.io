Билет 6. Бесконечно большие и бесконечно малые последовательности, их примеры, свойства. Предел суммы, разности, произведения и частного двух числовых последовательностей.

Ниже представлен максимально подробный ответ на экзаменационный билет, составленный на основе предоставленных материалов. Ответ структурирован так, чтобы логически объяснить каждое понятие «с нуля» и привести полные математические доказательства всех свойств.

1. Бесконечно малые последовательности (БМП)

Определение. Последовательность {αₙ} называется бесконечно малой, если её предел равен нулю. Формально: ∀ε > 0 ∃ Nₑ ∈ N : ∀ n > Nₑ ⟹ |αₙ| < ε. Это означает, что для любого, даже самого малого положительного числа ε, почти все члены последовательности (начиная с некоторого номера) находятся в ε-окрестности нуля.

Примеры:

  • αₙ = 1/n (основной пример БМП).
  • αₙ = (-1)ⁿ / n (знакочередующаяся БМП).
  • αₙ = 1/2ⁿ или αₙ = qⁿ при |q| < 1.

Свойства бесконечно малых последовательностей

  • Связь с пределом: Число a является пределом последовательности {xₙ} тогда и только тогда, когда разность αₙ = xₙ − a является бесконечно малой (xₙ = a + αₙ).
  • Сумма БМП: Сумма конечного числа БМП есть БМП.

    Доказательство: Пусть αₙ → 0 и βₙ → 0. Для произвольного ε > 0 найдутся номера N₁ и N₂ такие, что |αₙ| < ε/2 для n > N₁ и |βₙ| < ε/2 для n > N₂. Тогда для всех n > max(N₁, N₂) по неравенству треугольника: |αₙ + βₙ| ≤ |αₙ| + |βₙ| < ε/2 + ε/2 = ε. Ч.т.д.

  • Произведение БМП на ограниченную последовательность: Произведение бесконечно малой последовательности на любую ограниченную последовательность (в том числе на константу или другую БМП) является бесконечно малой последовательностью.

    Доказательство: Пусть αₙ → 0, а {yₙ} ограничена (|yₙ| ≤ C). Для ε > 0 возьмем номер N такой, что |αₙ| < ε/C. Тогда для n > N выполняется: |αₙ yₙ| = |αₙ|·|yₙ| < (ε/C)·C = ε. Ч.т.д.

2. Бесконечно большие последовательности (ББП)

Определение. Последовательность {xₙ} называется бесконечно большой, если её предел равен бесконечности (∞). Формально: ∀ E > 0 ∃ Nₑ : ∀ n > Nₑ ⟹ |xₙ| > E. Это означает, что члены последовательности по модулю со временем «пробивают» любую наперед заданную границу E.

  • Если xₙ > E, говорят, что lim xₙ = +∞.
  • Если xₙ < −E, говорят, что lim xₙ = −∞.

Примеры:

  • xₙ = n (стремится к +∞).
  • xₙ = (−1)ⁿ · n (стремится к ∞ без знака, члены колеблются).

Связь БМП и ББП: Если {αₙ} — БМП и все αₙ ≠ 0, то последовательность {1/αₙ} является бесконечно большой. И наоборот: если {xₙ} — ББП и xₙ ≠ 0, то {1/xₙ} — БМП.

Важное отличие: Любая бесконечно большая последовательность является неограниченной, но не всякая неограниченная последовательность является бесконечно большой.

Пример: xₙ = n·(1 − (−1)ⁿ) принимает значения (0, 2, 0, 4, 0, 6, …). Она неограничена, но не ББП, так как бесконечно много её членов равны нулю и не уходят на бесконечность.

3. Арифметические свойства пределов

Пусть даны две сходящиеся последовательности lim aₙ = a и lim bₙ = b. Тогда справедливы следующие свойства:

А) Предел суммы и разности

lim (aₙ ± bₙ) = a ± b

Доказательство: Представим последовательности через их пределы и бесконечно малые добавки: aₙ = a + αₙ, bₙ = b + βₙ, где αₙ, βₙ → 0. Тогда (aₙ + bₙ) = (a + αₙ) + (b + βₙ) = (a + b) + (αₙ + βₙ). Так как сумма двух БМП (αₙ + βₙ) есть БМП, то по свойству связи предела и БМП, предел суммы равен a + b. Ч.т.д.

Б) Предел произведения

lim (aₙ · bₙ) = a · b

Доказательство: Используем те же представления: aₙ · bₙ = (a + αₙ)(b + βₙ) = ab + aβₙ + bαₙ + αₙβₙ. Рассмотрим слагаемые:

  • ab — константа.
  • aβₙ — БМП (произведение константы a на БМП βₙ).
  • bαₙ — БМП (произведение константы b на БМП αₙ).
  • αₙβₙ — БМП (произведение двух БМП).

Сумма трех БМП (aβₙ + bαₙ + αₙβₙ) является БМП. Значит, lim (aₙ bₙ) = ab. Ч.т.д.

В) Предел частного

lim (aₙ / bₙ) = a / b, при условии b ≠ 0

Доказательство: Достаточно доказать, что lim (1/bₙ) = 1/b, а затем воспользоваться свойством произведения. Рассмотрим разность:

|1/bₙ − 1/b| = |(b − bₙ) / (b·bₙ)| = |βₙ| · (1 / |b·bₙ|)

Мы имеем произведение БМП (|βₙ|) на последовательность {1/|b·bₙ|}. По свойству БМП, нам нужно доказать, что {1/|b·bₙ|} ограничена. Так как b ≠ 0, то по лемме о сохранении знака, начиная с некоторого номера N, все |bₙ| > |b|/2. Тогда:

1 / |b·bₙ| < 1 / (|b|·|b|/2) = 2 / |b|²

Это константа, значит последовательность ограничена. Произведение БМП на ограниченную величину дает БМП, следовательно lim (1/bₙ) = 1/b. Итого:

lim (aₙ / bₙ) = lim (aₙ · 1/bₙ) = a · 1/b = a/b

Ч.т.д.
