Билет 12. Функции (отображения). Сюръективные, инъективные, биективные отображения. Обратное отображение, композиция отображений. Способы задания отображений (функций).

Ниже представлен подробный ответ по билету №12, составленный на основе предоставленных материалов.

1. Определение функции (отображения)

Понятие функции является одним из центральных в математике. Оно описывает правило, по которому элементы одного множества соотносятся с элементами другого.

Определение. Пусть даны два непустых множества X и Y. Говорят, что на множестве X задана функция (отображение) f со значениями в множестве Y, если по некоторому правилу каждому элементу x ∈ X поставлен в соответствие единственный элемент y ∈ Y.

Символически это записывается так: f: X → Y или y = f(x).

  • Множество X называется областью определения функции.
  • Элемент x ∈ X называется аргументом (или прообразом).
  • Элемент y = f(x) называется значением функции в точке x (или образом точки x).

Множество всех образов Y_f = f(X) = {y ∈ Y | ∃ x ∈ X, y = f(x)} называется множеством значений функции. Важно понимать, что множество значений Y_f может быть лишь подмножеством множества Y (Y_f ⊆ Y).

Ключевое свойство: Для функции принципиально важно, чтобы каждому x соответствовал ровно один y. Если одному и тому же аргументу соответствуют несколько значений, такое отношение функцией не является.

2. Классификация отображений

Отображения различаются по тому, как именно элементы множества X «покрывают» множество Y.

Инъективное отображение (инъекция)

Отображение называется инъективным, если разным элементам из X всегда соответствуют разные элементы в Y. Формально: ∀x₁, x₂ ∈ X, x₁ ≠ x₂ ⇒ f(x₁) ≠ f(x₂). Простыми словами: функция не «склеивает» разные точки. Пример: f(x) = x³ на всей прямой.

Сюръективное отображение (сюръекция)

Отображение называется сюръективным («отображением на»), если каждый элемент множества Y имеет хотя бы один прообраз в множестве X. В этом случае множество значений функции совпадает со всем множеством Y (Y_f = Y). Формально: ∀y ∈ Y ∃ x ∈ X : y = f(x). Простыми словами: функция «закрывает» своими значениями всё целевое множество Y.

Биективное отображение (биекция)

Отображение называется биективным (или взаимно однозначным соответствием), если оно одновременно является и инъективным, и сюръективным. Простыми словами: это идеальное соответствие «один к одному» между элементами X и Y. В этом случае множества называются равномощными.

3. Обратное отображение

Если функция f: X → Y является биекцией, то для неё можно определить обратное отображение f⁻¹: Y → X.

Определение. Обратным отображением называется такое правило, которое каждому элементу y ∈ Y ставит в соответствие тот единственный элемент x ∈ X, для которого f(x) = y. Формально: f⁻¹(y) = x ⟺ f(x) = y.

Для существования обратного отображения необходима биективность исходного отображения: сюръективность гарантирует, что для каждого y найдется прообраз, а инъективность — что этот прообраз будет единственным.

Если функция строго монотонна (только возрастает или только убывает), она обязательно инъективна, что часто используется для обоснования существования обратных функций в анализе.

4. Композиция отображений

Композиция (или сложная функция) — это последовательное применение двух или более отображений.

Определение. Пусть даны отображения f: X → Y и g: Y → Z. Композицией отображений f и g называется отображение h: X → Z, действующее по правилу: h(x) = g(f(x)). Обозначается: h = g ∘ f.

Важно, что для возможности построения композиции область значений первого отображения f должна входить в область определения второго отображения g. При расчете производных для таких функций применяется «цепное правило» (производная сложной функции).

5. Способы задания отображений

В математическом анализе используются различные методы описания правила соответствия:

  • Аналитический способ. Функция задается одной или несколькими формулами (например, f(x) = x² + 4x + 6). Это наиболее распространенный способ в анализе.
  • Словесный способ (правило). Описание правила словами. Пример: «функция Дирихле равна 1 для рациональных чисел и 0 для иррациональных».
  • Табличный способ. Перечисление пар аргумент-значение. Обычно применяется для конечных множеств или в численных методах.
  • Графический способ. Функция представляется в виде графика на плоскости или в пространстве. Геометрически это множество точек (x, f(x)).
  • Параметрический способ. Значения аргумента x и функции y зависят от некоторого вспомогательного параметра t: x = x(t), y = y(t).
  • Неявный способ. Функция задается уравнением вида Φ(x, y) = 0, которое связывает аргумент и значение, но не разрешено относительно y (например, уравнение окружности).
  • Задание в виде последовательности. Последовательность можно рассматривать как функцию, где областью определения является множество натуральных чисел N.
