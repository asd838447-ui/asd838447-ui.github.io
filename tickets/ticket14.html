Ниже представлен подробный разбор билета №14, составленный на основе предоставленных источников. Ответ написан максимально развернуто, с пояснениями всех терминов и полными доказательствами.

1. Переход к пределу в неравенствах

Эта теорема связывает поведение значений функций с поведением их пределов. Она утверждает, что если одна функция не превосходит другую в окрестности точки, то и её предел не будет больше предела другой функции.

Теорема (о предельном переходе в неравенствах): Пусть функции f(x) и g(x) определены на множестве I, и существуют их конечные пределы в точке a: lim_{x→a} f(x) = b и lim_{x→a} g(x) = d. Если в некоторой проколотой окрестности точки a выполняется неравенство f(x) ≤ g(x), то и для пределов выполняется неравенство: b ≤ d.

Доказательство: Доказательство удобнее всего провести через определение предела по Гейне (на языке последовательностей).

  1. Рассмотрим произвольную последовательность {xₙ}, сходящуюся к a (xₙ → a), где все xₙ ∈ I и xₙ ≠ a.
  2. По определению Гейне, если lim_{x→a} f(x) = b и lim_{x→a} g(x) = d, то соответствующие последовательности значений функций сходятся к этим числам: f(xₙ) → b и g(xₙ) → d при n → ∞.
  3. По условию, начиная с некоторого момента (как только xₙ попадут в нужную окрестность точки a), для членов последовательностей будет выполняться неравенство f(xₙ) ≤ g(xₙ).
  4. Из теории пределов последовательностей известно: если члены одной сходящейся последовательности не превосходят соответствующих членов другой, то и предел первой не превосходит предела второй. Следовательно, b ≤ d.

Важное замечание: Даже если исходное неравенство строгое (f(x) < g(x)), при переходе к пределу оно может превратиться в равенство. Например, f(x) = 0, а g(x) = x² при x → 0. Здесь g(x) > f(x) везде, кроме нуля, но их пределы в нуле равны 0.

2. Лемма о сохранении знака

Эта лемма говорит о том, что если функция имеет в точке предел, отличный от нуля, то в достаточно малой окрестности этой точки значения функции будут иметь тот же знак, что и сам предел.

Лемма (о знаке): Пусть lim_{x→a} f(x) = b и b ≠ 0. Тогда существует такая окрестность точки a, в которой для всех x из области определения значения функции f(x) имеют тот же знак, что и число b. Более точно:

  • Если b > 0, то существует U(a, δ), где f(x) > b/2 > 0.
  • Если b < 0, то существует U(a, δ), где f(x) < b/2 < 0.

Доказательство (для случая b > 0):

  1. Воспользуемся определением предела по Коши. Возьмем в качестве положительного числа ε величину |b|/2. Поскольку b > 0, то ε = b/2.
  2. По определению предела, для этого ε найдется такая окрестность U(a, δ), что для всех x из неё выполняется неравенство: |f(x) - b| < b/2.
  3. Раскроем модуль: -b/2 < f(x) - b < b/2.
  4. Прибавим b ко всем частям неравенства: b - b/2 < f(x) < b + b/2 ⟹ b/2 < f(x) < 3b/2.
  5. Таким образом, во всей этой окрестности f(x) > b/2. Так как b > 0, то и b/2 > 0, значит, значения функции положительны. Аналогично доказывается для случая b < 0.

3. Теорема о зажатой функции (Теорема о «двух милиционерах»)

Эта теорема позволяет найти предел функции, которую сложно вычислить напрямую, если удается «зажать» её между двумя другими функциями, имеющими одинаковый предел.

Теорема: Пусть три функции h(x), f(x) и g(x) определены в некоторой проколотой окрестности точки a. Если для всех x из этой окрестности выполняется неравенство: h(x) ≤ f(x) ≤ g(x) и при этом пределы крайних функций в точке a существуют и равны одному и тому же числу p: lim_{x→a} h(x) = lim_{x→a} g(x) = p, то предел функции f(x) в этой точке также существует и равен p.

Доказательство (через определение Гейне):

  1. Возьмем произвольную последовательность точек Гейне {xₙ}, такую что xₙ → a при n → ∞.
  2. По условию теоремы, для членов этой последовательности будет выполняться неравенство: h(xₙ) ≤ f(xₙ) ≤ g(xₙ).
  3. Поскольку lim_{x→a} h(x) = p и lim_{x→a} g(x) = p, то по определению Гейне последовательности значений h(xₙ) и g(xₙ) обе стремятся к числу p.
  4. По лемме о зажатой последовательности (для числовых последовательностей), если последовательность f(xₙ) находится между двумя другими, стремящимися к p, то она сама стремится к p.
  5. Так как это верно для любой последовательности Гейне, по определению предела функции по Гейне lim_{x→a} f(x) = p.

Геометрический смысл: Если две функции h(x) и g(x) «загоняют» функцию f(x) в одну и ту же «дверь» (значение A) при приближении аргумента к x₀, то функции f(x) некуда деться, кроме как прийти в ту же точку.

4. Бесконечно малые функции и их свойства

Определение: Функция α(x) называется бесконечно малой (б.м.ф.) при x → a, если её предел в этой точке равен нулю: lim_{x→a} α(x) = 0.

Связь с конечным пределом: Функция f(x) имеет предел b при x → a тогда и только тогда, когда её можно представить в виде суммы числа b и бесконечно малой функции γ(x): f(x) = b + γ(x), где lim_{x→a} γ(x) = 0.

Основные свойства б.м.ф.:

  • Сумма: Алгебраическая сумма конечного числа бесконечно малых функций есть функция бесконечно малая (следует из свойств предела суммы).
  • Произведение на ограниченную функцию: Произведение бесконечно малой функции на функцию, ограниченную в окрестности точки a, является бесконечно малой функцией.
  • Произведение: Произведение двух (и более) бесконечно малых функций есть бесконечно малая функция (как частный случай произведения на ограниченную).
  • Связь с бесконечно большими: Если α(x) — бесконечно малая при x → a и не обращается в ноль в окрестности a, то функция 1/α(x) является бесконечно большой при x → a.

5. Формулировка критерия Коши о пределе функции

Критерий Коши важен тем, что позволяет установить существование предела функции, не зная самого значения этого предела заранее.

Теорема (Критерий Коши): Для того чтобы функция f(x) имела конечный предел в точке a, необходимо и достаточно, чтобы для любого ε > 0 существовала такая проколотая окрестность \dot{U}(a, δ), что для любых двух точек x' и x'' из этой окрестности выполнялось неравенство: |f(x') - f(x'')| < ε.

Краткая схема доказательства:

Необходимость: Пусть lim_{x→a} f(x) = b. По определению предела, для любого ε/2 > 0 найдется окрестность \dot{U}(a, δ), в которой |f(x) - b| < ε/2. Тогда для любых x', x'' из этой окрестности: |f(x') - f(x'')| = |(f(x') - b) - (f(x'') - b)| ≤ |f(x') - b| + |f(x'') - b| < ε/2 + ε/2 = ε.

Достаточность: Пусть условие Коши выполнено. Возьмем любую последовательность Гейне xₙ → a. Тогда последовательность значений f(xₙ) будет фундаментальной (последовательностью Коши) и, в силу полноты множества действительных чисел, будет иметь предел. С помощью метода «смешанной последовательности» доказывается, что для всех последовательностей Гейне этот предел будет одним и тем же.

6. Теорема о пределе композиции функций (без доказательства)

Эта теорема описывает, как вычислять предел сложной функции g(f(x)).

Теорема (о пределе сложной функции): Пусть существуют:

  • Предел внутренней функции: lim_{x→a} f(x) = b.
  • Внешняя функция g(y) непрерывна в точке b.

Тогда предел сложной функции g(f(x)) существует и равен значению внешней функции от предела внутренней:

lim_{x→a} g(f(x)) = g(b)

Суть: Если внешняя функция непрерывна, знак предела можно вносить под знак функции: lim g(f(x)) = g(lim f(x)).
