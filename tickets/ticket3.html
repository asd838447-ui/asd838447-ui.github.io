Билет 3. Модуль числа, два простых неравенства с модулем. Системы вложенных отрезков, стягивающиеся системы вложенных отрезков, две теоремы об этих системах.

Этот билет посвящен изучению фундаментальных понятий математического анализа: модулю действительного числа и его свойствам, а также принципу вложенных отрезков, который является одной из форм аксиомы непрерывности действительных чисел.

1. Модуль действительного числа

Определение. Модулем (или абсолютной величиной) действительного числа a (обозначается |a|) называется само это число, если оно неотрицательно, и это же число с противоположным знаком, если оно отрицательно. Математически это записывается так:

|a| = { a, если a ≥ 0; −a, если a < 0 }

Геометрический смысл: Модуль числа a отражает расстояние от начала координат (точки 0) до точки a на числовой прямой. Поскольку расстояние не может быть отрицательным, модуль любого числа всегда неотрицателен: |a| ≥ 0.

Полезные эквивалентные свойства:

  • |a| = max(−a, a).
  • Справедливы неравенства: −a ≤ |a| и a ≤ |a|.

2. Два простых неравенства с модулем

Рассмотрим два базовых неравенства, которые постоянно используются в доказательствах теорем о пределах и непрерывности функций.

Неравенство 1 (Неравенство треугольника)

Для любых действительных чисел a и b справедливо:

|a + b| ≤ |a| + |b|

Доказательство: Оно основывается на свойствах упорядоченности действительных чисел.

  1. Мы знаем, что для любых чисел выполняются соотношения: a ≤ |a| и b ≤ |b|. Сложив их, получаем: a + b ≤ |a| + |b|.
  2. Также верны соотношения: −a ≤ |a| и −b ≤ |b|. Сложив их, получаем: −(a + b) ≤ |a| + |b|.
  3. Поскольку модуль суммы |a + b| равен либо (a + b), либо −(a + b), а мы показали, что обе эти величины не превосходят сумму |a| + |b|, то и сам модуль не превосходит её: max(−(a + b), a + b) ≤ |a| + |b|.

Неравенство 2 (Неравенство для разности модулей)

Для любых действительных чисел a и b справедливо:

||a| − |b|| ≤ |a − b|

Доказательство:

  1. Представим число a в виде суммы: a = (a − b) + b. Применим к этой сумме неравенство треугольника (доказанное выше): |a| = |(a − b) + b| ≤ |a − b| + |b|. Отсюда следует: |a| − |b| ≤ |a − b|.
  2. Теперь проделаем то же самое для числа b, представив его как b = (b − a) + a: |b| = |(b − a) + a| ≤ |b − a| + |a|. Учитывая, что |b − a| = |a − b|, получаем: |b| − |a| ≤ |a − b|, что равносильно −(|a| − |b|) ≤ |a − b|.
  3. Так как и величина (|a| − |b|), и её отрицание −(|a| − |b|) не превышают |a − b|, то и модуль их разности ||a| − |b|| не превышает |a − b|.

3. Системы вложенных отрезков

В математическом анализе часто рассматривают не просто наборы чисел, а последовательности целых промежутков (отрезков).

Определение. Система числовых отрезков {[aₙ, bₙ]} (где n = 1, 2, …) называется системой вложенных отрезков (СВО), если каждый последующий отрезок полностью содержится в предыдущем. Формально: [aₙ₊₁, bₙ₊₁] ⊂ [aₙ, bₙ] для всех n ∈ N.

Важные следствия из определения:

  • Последовательность левых концов {aₙ} является неубывающей: a₁ ≤ a₂ ≤ … ≤ aₙ ≤ …
  • Последовательность правых концов {bₙ} является невозрастающей: b₁ ≥ b₂ ≥ … ≥ bₙ ≥ …
  • Любой левый конец никогда не превосходит любого правого конца: aₖ ≤ bₙ для любых k, n.

4. Стягивающиеся системы вложенных отрезков

Это частный случай вложенных отрезков, когда их длина становится бесконечно малой.

Определение. Система вложенных отрезков {[aₙ, bₙ]} называется стягивающейся, если для любого сколь угодно малого положительного числа ε > 0 найдется такой номер nₑ, что длина соответствующего отрезка будет меньше этого числа: bₙₑ − aₙₑ < ε. Другими словами, длина отрезков стремится к нулю: lim (bₙ − aₙ) = 0 при n→∞.

5. Две теоремы о системах вложенных отрезков

Эти теоремы (часто объединяемые под названием «Принцип вложенных отрезков Кантора») описывают структуру пересечения таких отрезков.

Теорема 1 (О существовании общей точки)

Формулировка: Всякая система вложенных отрезков имеет непустое пересечение. То есть существует по крайней мере одна точка c, общая для всех отрезков системы.

Доказательство:

  1. Рассмотрим два множества: множество левых концов A = {aₙ} и множество правых концов B = {bₙ}.
  2. Как было отмечено выше, любой левый конец не превосходит любого правого конца: aₖ ≤ bₙ для всех k, n.
  3. Согласно аксиоме непрерывности (свойству полноты) действительных чисел, если для двух непустых множеств A и B выполняется условие a ≤ b для всех a ∈ A, b ∈ B, то существует такое число c, которое их разделяет: aₙ ≤ c ≤ bₙ для всех n.
  4. Неравенство aₙ ≤ c ≤ bₙ означает, что точка c принадлежит каждому отрезку [aₙ, bₙ] по определению отрезка. Значит, c — общая точка.

Теорема 2 (О единственности точки в стягивающейся системе)

Формулировка: Если система вложенных отрезков является стягивающейся, то существует единственная точка, принадлежащая всем отрезкам этой системы.

Доказательство:

  1. Существование хотя бы одной общей точки c уже доказано в Теореме 1.
  2. Докажем единственность методом от противного. Предположим, что существуют две различные общие точки c и d, причем c ≠ d. Пусть для определенности d > c, тогда расстояние между ними |d − c| = L > 0.
  3. Поскольку обе точки принадлежат всем отрезкам, то для каждого n выполняется aₙ ≤ c < d ≤ bₙ. Длина любого отрезка bₙ − aₙ в этом случае должна быть не меньше расстояния между точками: bₙ − aₙ ≥ d − c = L.
  4. Однако по определению стягивающейся системы, длины отрезков стремятся к нулю. Это значит, что для любого ε > 0, включая ε < L, должен найтись отрезок длиной меньше ε.
  5. Мы получили противоречие: длина отрезка должна быть одновременно и меньше ε, и больше или равна L. Следовательно, двух разных точек быть не может, и общая точка единственна.
