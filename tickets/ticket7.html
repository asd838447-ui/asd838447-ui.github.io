Билет 7. Монотонные последовательности. Теорема Вейерштрасса об ограниченной монотонной последовательности. Число e.

В данном билете мы подробно рассмотрим одну из самых важных тем математического анализа: поведение монотонных последовательностей, критерий их сходимости (теорему Вейерштрасса) и фундаментальную константу — число e.

1. Понятие монотонных последовательностей

Монотонность — это свойство последовательности сохранять характер изменения своих членов.

Определения:

  • Возрастающая (неубывающая): Последовательность {xₙ} называется возрастающей, если каждый её последующий член не меньше предыдущего: ∀n ∈ N : xₙ₊₁ ≥ xₙ.
  • Строго возрастающая: Если каждый следующий член строго больше предыдущего: ∀n ∈ N : xₙ₊₁ > xₙ.
  • Убывающая (невозрастающая): Последовательность {xₙ} называется убывающей, если каждый её последующий член не больше предыдущего: ∀n ∈ N : xₙ₊₁ ≤ xₙ.
  • Строго убывающая: Если каждый следующий член строго меньше предыдущего: ∀n ∈ N : xₙ₊₁ < xₙ.

Все такие последовательности называются монотонными.

2. Теорема Вейерштрасса о сходимости монотонной последовательности

Это одна из центральных теорем анализа. Она утверждает, что для сходимости монотонной последовательности к конечному числу достаточно лишь её ограниченности.

Формулировка:

  • Если последовательность {xₙ} возрастает и ограничена сверху, то она имеет конечный предел, равный её точной верхней грани (sup xₙ).
  • Если последовательность {xₙ} убывает и ограничена снизу, то она имеет конечный предел, равный её точной нижней грани (inf xₙ).

Доказательство (для случая возрастающей последовательности):

  1. Пусть последовательность {xₙ} возрастает (xₙ₊₁ ≥ xₙ) и ограничена сверху (существует число C, такое что xₙ ≤ C для всех n).
  2. Рассмотрим множество всех значений последовательности. Так как оно непусто и ограничено сверху, то по аксиоме полноты (непрерывности) действительных чисел у него существует точная верхняя грань. Обозначим её M = sup{xₙ}.
  3. Напомним два свойства супремума:
    • ∀n ∈ N : xₙ ≤ M (это верхняя грань).
    • ∀ε > 0 ∃ N ∈ N : x_N > M − ε (это наименьшая из верхних граней).
  4. Возьмем произвольное ε > 0. Согласно второму свойству, найдется такой номер N, что член последовательности x_N попадет в интервал (M − ε, M].
  5. Поскольку наша последовательность возрастает, то для всех номеров n ≥ N выполняется неравенство xₙ ≥ x_N.
  6. Следовательно, для всех n ≥ N мы получаем цепочку неравенств: M − ε < x_N ≤ xₙ ≤ M < M + ε.
  7. Это означает, что для всех n ≥ N выполняется |xₙ − M| < ε.
  8. По определению предела последовательности это в точности означает, что lim xₙ = M.

Замечание: Если возрастающая последовательность не ограничена сверху, то её предел равен +∞.

3. Число e

Число e определяется как предел последовательности aₙ = (1 + 1/n)ⁿ. Чтобы доказать существование этого предела, мы должны показать, что данная последовательность монотонно возрастает и ограничена сверху, после чего применить теорему Вейерштрасса.

3.1 Вспомогательные инструменты для доказательства

  • Неравенство Бернулли: Для любого действительного числа a ≥ −1 и любого натурального n справедливо неравенство (1 + a)ⁿ ≥ 1 + na.
  • Теорема Вейерштрасса: Всякая возрастающая последовательность, ограниченная сверху, имеет предел, равный её точной верхней грани.

3.2 Доказательство монотонного возрастания последовательности aₙ = (1 + 1/n)ⁿ

Рассмотрим отношение последующего члена последовательности к предыдущему: aₙ / aₙ₋₁. Наша цель — показать, что это отношение всегда больше или равно 1, что будет означать, что каждый следующий член не меньше предыдущего.

aₙ / aₙ₋₁ = (1 + 1/n)ⁿ / (1 + 1/(n−1))ⁿ⁻¹

Преобразуем это выражение:

aₙ / aₙ₋₁ = ( (n+1)/n )ⁿ / ( n/(n−1) )ⁿ⁻¹ = (n/(n−1)) · ( (n+1)/n · (n−1)/n )ⁿ = (n/(n−1)) · ( (n² − 1)/n² )ⁿ = (n/(n−1)) · (1 − 1/n²)ⁿ

Применим к выражению в скобках неравенство Бернулли, где роль a играет (−1/n²):

(1 − 1/n²)ⁿ ≥ 1 − n·(1/n²) = 1 − 1/n = (n−1)/n

Подставим полученную оценку обратно в наше отношение:

aₙ / aₙ₋₁ ≥ (n/(n−1)) · ((n−1)/n) = 1

Так как aₙ / aₙ₋₁ ≥ 1, мы доказали, что последовательность aₙ монотонно возрастает (aₙ ↑).

3.3 Доказательство ограниченности

Для доказательства ограниченности сверху введем вспомогательную последовательность bₙ = (1 + 1/n)ⁿ⁺¹. Очевидно, что для любого n выполняется неравенство aₙ < bₙ.

Докажем теперь, что последовательность bₙ монотонно убывает. Для этого рассмотрим отношение bₙ₋₁ / bₙ:

bₙ₋₁ / bₙ = (1 + 1/(n−1))ⁿ / (1 + 1/n)ⁿ⁺¹ = ... = (n/(n−1)) · (1 + 1/(n² − 1))ⁿ⁺¹

Снова применяем неравенство Бернулли:

(1 + 1/(n² − 1))ⁿ⁺¹ ≥ 1 + (n+1)/(n² − 1) = 1 + (n+1)/((n−1)(n+1)) = 1 + 1/(n−1) = n/(n−1)

Подставляем в отношение:

bₙ₋₁ / bₙ ≥ (n/(n−1)) · (n/(n−1)) = (n/(n−1))² ≥ 1

Следовательно, bₙ₋₁ ≥ bₙ, то есть bₙ убывает.

3.4 Итоговая оценка ограниченности

Так как aₙ возрастает, а bₙ убывает, и при этом aₙ < bₙ для любого n, мы можем записать:

2 = a₁ ≤ aₙ < bₙ ≤ b₁ = 4

Это означает, что наша основная последовательность aₙ всегда находится в пределах от 2 до 4. Следовательно, она ограничена сверху.

3.5 Определение числа e

Поскольку мы доказали, что последовательность aₙ = (1 + 1/n)ⁿ:

  • Монотонно возрастает;
  • Ограничена сверху (числом 4),

то по теореме Вейерштрасса она имеет конечный предел. Этот предел и называется числом e:

e = lim (1 + 1/n)ⁿ при n → ∞

Также можно показать, что предел убывающей последовательности bₙ совпадает с e:

lim bₙ = lim (aₙ · (1 + 1/n)) = (lim aₙ) · (lim (1 + 1/n)) = e · 1 = e

Число e — это фундаментальная математическая константа, иррациональное число, приближенно равное 2,71828... Оно является точной верхней гранью (sup) последовательности aₙ и точной нижней гранью (inf) последовательности bₙ.
